19.1

結構與鍵結

羧酸（carboxylic acid）含有一個 羧基（carboxy group， –COOH）。這個名字來自 carbonyl（C=O）+ hydroxy（OH）的組合。

羧基中央的碳原子同時與一個氧形成 雙鍵（C=O），並與另一個氧形成 單鍵（C–OH）。縮寫寫法為 RCOOH 或 RCO₂H。
混成軌域
sp²
羧基碳被三組原子包圍 → 三角平面，鍵角約 120°
C=O 鍵長
121 pm
C–O 單鍵為 136 pm；雙鍵較短，符合更高鍵級

乙酸的 C–OH 單鍵（136 pm）比醇的 C–O 單鍵（143 pm）還短，原因是羧基碳是 sp² 雜化（s 特性較高），醇的碳是 sp³ 雜化（s 特性較低）。

Figure 19.1｜乙酸的靜電位能圖
圖中以顏色代表電荷分布：
• 🔴 紅色區域（兩個氧原子）= 電子密度高（負電）
• 🔵 藍色區域（羰基碳、OH 的氫）= 電子密度低（正電）
這說明 O 比 C 和 H 更具電負性，導致鍵結有極性。

命名法

A. IUPAC 命名

1
若 COOH 接在碳鏈上 → 找最長含 COOH 的碳鏈，將母烷烴的 -e 改成 -oic acid
若 COOH 接在環上 → 命名環，再加 carboxylic acid
2
編號使 COOH 在 C1（省略數字 1），其餘取代基依一般規則命名
COOH 本身的碳也算在最長碳鏈中！例如 hexanoic acid（6碳）中，第1個碳就是 COOH 的碳。

19.5

有趣的羧酸

🐜 甲酸（formic acid，HCOOH）
某些螞蟻螫人時釋放的酸，名字來自拉丁文 formica（螞蟻），有嗆鼻氣味。
🍶 乙酸（acetic acid，CH₃COOH）
醋的酸味來源。乙醇暴露在空氣中被氧化即生成乙酸，讓酒變酸。純乙酸又稱冰醋酸（glacial acetic acid），熔點 17°C，常溫下可凝固。
😰 丁酸（butyric acid，CH₃CH₂CH₂COOH）
體臭和腐敗奶油的氣味來源，名字來自拉丁文 butyrum（奶油）。
🌿 草酸 & 乳酸
草酸（oxalic acid，(COOH)₂）存在菠菜和大黃中；
乳酸（lactic acid，CH₃CH(OH)COOH）使酸牛奶有特殊酸味，人體無氧運動也會生成乳酸。

防腐劑中的羧酸鹽

羧酸的鈉鹽／鉀鹽常用作食品防腐劑：

  • 苯甲酸鈉（sodium benzoate）：抑制真菌，用於軟性飲料
  • 山梨酸鉀（potassium sorbate）：延長烘焙食品等保存期限
19.6

阿斯匹靈、花生四烯酸與前列腺素

阿斯匹靈（aspirin）如何止痛？ → 阻斷前列腺素的合成！

阿斯匹靈（acetylsalicylic acid）是一個合成的羧酸，由 Felix Hoffmann 於 1899 年首次商業化合成（其父罹患類風濕性關節炎，促使他研發）。

止痛機制：
阿斯匹靈 → 抑制環氧合酶（cyclooxygenase）→ 阻斷「花生四烯酸（arachidonic acid）→ PGG₂ → 前列腺素」的合成路徑

前列腺素（prostaglandins）是負責疼痛和發炎的 20 碳脂肪酸（含五元環）。
阿斯匹靈相關化合物
• 水楊酸（salicylic acid）：從草地芥末（meadowsweet）中分離，但會刺激口腔和胃黏膜
• 水楊苷（salicin）：從柳樹皮中分離，名稱 aspirin 來自 a-（乙醯基）+ spir-（meadowsweet 的拉丁屬名 spirea）
• 拉他諾前列素（latanoprost）／貝美前列素（bimatoprost）：前列腺素類似物，用於治療青光眼（降低眼壓）
19.7

羧酸的製備

羧基碳高度氧化（有三個 C–O 鍵），因此羧酸通常由氧化反應製備。

三種主要氧化製備法
1
一級醇（1° alcohol）的氧化
試劑：Na₂Cr₂O₇、K₂Cr₂O₇ 或 CrO₃（在 H₂SO₄ / H₂O 中）
R–CH₂OH ‒‒[K₂Cr₂O₇， H₂SO₄]‒→ R–COOH
注意：二級醇只會氧化到酮，不會繼續生成羧酸
2
烷基苯（alkyl benzene）的氧化
試劑：KMnO₄（苯環側鏈至少有一個苄位 C–H）
PhCH₂R（或 PhCH₂CH₃ 等）‒‒KMnO₄‒→ PhCOOH（苯甲酸）
無論側鏈多長，產物永遠是苯甲酸（benzoic acid）!
3
炔烴（alkyne）的氧化裂解
試劑：O₃，然後加水（H₂O）
內部炔烴 ‒‒[1] O₃， [2] H₂O‒→ 兩個羧酸
末端炔烴 ‒‒[1] O₃， [2] H₂O‒→ 一個羧酸 + CO₂
19.8

羧酸的反應概觀

羧酸有多個反應位點：

最重要 ⭐
O–H 鍵斷裂
被鹼奪走質子 → Brønsted-Lowry 酸反應（本章重點）
次要 ①
O 上的孤對電子
被強酸質子化（在羰基氧上）→ 羧酸弱鹼性（第22章重要）
次要 ②
親核攻擊
親核試劑攻擊電正性的羰基碳 → π 鍵斷裂（第22章）
質子化發生在羰基氧（C=O 的 O），而非 OH 的氧，因為質子化後的共振穩定結構更多，使產物更穩定。
19.9

羧酸 — 強有機 Brønsted-Lowry 酸

羧酸為什麼酸性強？因為它的共軛鹼（羧酸根陰離子）被共振高度穩定化！

羧酸和鹼反應，生成羧酸根陰離子（carboxylate anion）：

R–COOH + :B ⇌ R–COO⁻ + H–B⁺
乙酸根（acetate）的共振結構：
CH₃COO⁻ 有兩個等值共振結構，負電荷均等分布在兩個氧上。
實驗證明：醋酸根的兩個 C–O 鍵長相同，均為 127 pm（介於 C–O 單鍵 136 pm 和 C=O 121 pm 之間），印證了共振雜化。
為什麼羧酸比醇和酚更酸？

pKₐ = 16
最弱酸
乙氧基（ethoxide）無額外共振穩定
苯酚（phenol）
pKₐ = 10
中等酸
苯氧基有5個共振結構，但部分負電荷在碳（電負性較小）
乙酸（RCOOH）
pKₐ = 4.8
最強有機酸 ⭐
乙酸根有2個等值共振結構，負電荷完全在兩個電負性氧原子上
Figure 19.7｜乙酸、苯酚、乙醇酸性比較總結
酸性（pKₐ）↑ 大 = 酸性弱，共軛鹼 ↓ 不穩定
• 乙酸根（acetate）：最穩定（兩個等值共振，都在 O 上）
• 苯氧基（phenoxide）：次穩定（5個共振，但部分在碳上，且有芳香性損失）
• 乙氧基（ethoxide）：最不穩定（無額外共振）

常用的脫質子試劑

對 pKₐ ≈ 5 的羧酸，需使用共軛酸 pKₐ > 5 的鹼才能有效脫質子：

（苯酚 pKₐ=10，需要 pKₐ>10 的鹼；乙醇 pKₐ=16，只有 NaH 夠強）

脂肪族羧酸中的誘導效應

「靠近 COOH 的基團能幫忙穩定負電荷，酸性就更強！」
電子提供基（electron-donating groups）
如烷基（R–）推電子→ 使共軛鹼更不穩定 → 酸性降低
例：（CH₃)₃CCOOH，pKₐ = 5.1（比乙酸弱）
電子吸引基（electron-withdrawing groups）
如鹵素（Cl， F）拉電子→ 使共軛鹼更穩定 → 酸性增強
例：ClCH₂COOH，pKₐ = 2.8（比乙酸強）

19.11

取代苯甲酸

苯環上的取代基既影響親電子芳香取代反應，也影響苯甲酸衍生物的酸性。

規則一：電子提供基（D）
推電子給羧酸根 → 不穩定 → 酸性降低
pKₐ > 4.2（苯甲酸的 pKₐ）

常見例子：–NH₂、–OH、–OR、–R
規則二：電子吸引基（W）
從羧酸根拉走電子 → 穩定 → 酸性增強
pKₐ < 4.2（苯甲酸的 pKₐ）

常見例子：–NO₂、–CN、–COOH、–X（鹵素）
記憶連結（與第18章一致）：
• 活化苯環（electrophilic aromatic substitution 反應快）的基團 = 電子提供基 → 使苯甲酸較不酸
• 去活化苯環的基團 = 電子吸引基 → 使苯甲酸較酸
Figure 19.8｜取代基對苯甲酸酸性的影響（對應親電芳香取代反應）
例：
• p-甲氧基苯甲酸（p-methoxybenzoic acid）：–OCH₃ 為電子提供基 → pKₐ > 4.2（較苯甲酸弱）
• p-硝基苯甲酸（p-nitrobenzoic acid）：–NO₂ 為電子吸引基 → pKₐ < 4.2（較苯甲酸強）
排列酸性強弱：p-甲氧基苯甲酸 < 苯甲酸 < p-硝基苯甲酸
19.12

萃取（Extraction）

萃取是利用化合物在兩種不相溶溶劑（水層 vs 有機層）中的溶解度差異，來分離混合物的技術。使用「分液漏斗（separatory funnel）」來操作。

Figure 19.9｜分液漏斗的使用
兩種不相溶液體加入分液漏斗後，密度較低的在上層，密度較高的在下層。
打開下方旋塞即可分離下層液體，上層則從上口倒出。
常用有機溶劑：二氯甲烷（CH₂Cl₂）、乙醚、己烷（密度皆小於水）
如何用萃取分離苯甲酸（PhCOOH）和環己醇（cyclohexanol）？

兩者都溶於有機溶劑、不溶於水，直接萃取無法分離。但利用酸鹼化學就可以：

Figure 19.10｜萃取分離步驟
Step 1：兩種化合物都溶解在 CH₂Cl₂ 中
Step 2：加入 10% NaOH 溶液 →
  苯甲酸（pKₐ=4.2）被 NaOH 脫質子 → 苯甲酸鈉（PhCOO⁻Na⁺）溶入水層
  環己醇（pKₐ≈17）太弱，NaOH 無法脫質子 → 留在有機層
Step 3：打開旋塞分離兩層 → 分離完成！

✔ 若要回收苯甲酸：對水層加強酸（HCl）使 PhCOO⁻ 重新質子化 → PhCOOH 析出
19.13

磺酸（Sulfonic Acids）

磺酸的通式為 R–SO₃H，比羧酸少見，但非常重要。最常用的是 對甲苯磺酸（p-toluenesulfonic acid，TsOH）。

磺酸為何是強酸？
pKₐ ≈ –7（遠比羧酸的 ~5 強！接近強無機酸）
原因：共軛鹼 磺酸根陰離子（sulfonate anion，RSO₃⁻）有 三個共振結構，負電荷均等分散在三個氧原子上，極度穩定！

磺酸根是很好的離去基（leaving group），因為它是極弱的鹼（共軛酸 pKₐ 很低），在親核取代反應中非常有用（見第9章）。

19.14

胺基酸（Amino Acids）

胺基酸 = 同時含有「羧酸（–COOH）」和「胺基（–NH₂）」的兩性化合物

A. 結構簡介

大多數天然胺基酸，胺基連在 α 碳上，稱為 α-胺基酸。通式：

H₂N–C*H(R)–COOH （* = 立體中心，除 glycine 外）
人體共需 20 種胺基酸，但只能自行合成其中 10 種。另外 10 種稱為必需胺基酸（essential amino acids），必須從飲食中攝取（幾乎每天）。

• 穀類（wheat， rice， corn）含少量離胺酸（lysine）
• 豆類含少量甲硫胺酸（methionine）
✅ 搭配食用即可獲得全部必需胺基酸！（例：米飯 +豆腐、玉米薄餅 + 豆子）

除 glycine（R=H）外，所有天然胺基酸的 α 碳都有立體中心，且自然界中只存在 L-異構物（大多為 S 構形，cysteine 例外）。

• pH ≤ 2：Form B（+1 charge）——羧基被質子化
• pH ≈ 6：Form A（zwitterion，中性）——兩性離子，這才是真正的「中性形態」
• pH ≥ 10：Form C（−1 charge）——胺基被去質子化

胺基酸固態、高熔點、可溶於水（因為是鹽類），不溶於有機溶劑（如乙醚）

C. 等電點（Isoelectric Point，pI）

等電點 pI：胺基酸主要以中性兩性離子形式存在的 pH 值

計算公式：
pI = [pKₐ(COOH) + pKₐ(NH₃⁺)] / 2
例：丙胺酸的 pKₐ(COOH) = 2.35，pKₐ(NH₃⁺) = 9.87
→ pI = (2.35 + 9.87) / 2 = 6.12

三、影響酸性的因素
共振效應
負電荷越分散（共振結構越穩定）→ 酸性越強
羧酸根 > 苯氧基 > 烷氧基
誘導效應（脂肪族）
電負性吸電基（Cl， F）→ 酸性增強
烷基供電基 → 酸性降低
距離越近，效果越大
苯甲酸衍生物
活化基（供電）→ pKₐ > 4.2（酸性弱）
去活化基（吸電）→ pKₐ < 4.2（酸性強）
四、胺基酸pH形態記憶口訣
「低 pH 帶正電，高 pH 帶負電，isoelectric point 淨電荷為零」

pH 很低（≤ 2）：加了很多 H⁺ → COOH 不離子化 → 整體帶 +1
pH ≈ pI（等電點）：兩性離子，淨電荷 = 0
pH 很高（≥ 10）：OH⁻ 把 NH₃⁺ 的質子拿走 → 整體帶 −1

pI = [pKₐ(COOH) + pKₐ(NH₃⁺)] / 2
