23.2 烯醇互變異構（Enols）

互變異構（tautomerism）是同一分子的兩種結構，差別只在一個質子（H⁺）和一個雙鍵的位置。
酮式（keto）和烯醇（enol）是最常見的一對互變異構體。

1

2

1

C(=O)-CH₂H + ⁻OH → [烯醇陰離子] + H₂O
2
質子化（protonation）：H₂O 將 H⁺ 給氧，形成烯醇
[O⁻-C=C] + H₂O → [HO-C=C]（烯醇）
鹼的口訣：先拿走 α-H，再加 H 到 O 上

烯醇如何反應？

核心概念
烯醇是富電子分子（electron-rich），α 碳因 O 的共振效應帶有負電，所以烯醇扮演親核劑，從 α 碳攻擊親電子試劑 E⁺。
淨反應後，α 碳上的 H 被 E 取代。

酸性規律 — 必背！

  • 負電荷在 O 上 比在 C 上更穩定 → 醇比烷更酸
  • 兩個 C=O 夾住的 CH₂（β-二羰基） 的 α-H 最酸！（負電荷可分散到兩個 O）

烯醇陰離子的反應性：雙面親核劑

Ambident Nucleophile（雙面親核劑）
烯醇陰離子有兩個反應位點：O 和 C（α 碳）都帶負電荷。

理論上可從 O 或 C 反應，但實際上幾乎從 C（α 碳）反應，因為 α 碳的親核性較強。

23.4 不對稱羰基化合物的烯醇陰離子

當酮的兩側 α 碳不同（例如 2-甲基環己酮），去質子化可以產生兩種不同的烯醇陰離子。

2-甲基環己酮的兩種可能烯醇陰離子
🏃 動力學烯醇陰離子
取走 二級 H（2° H）
→ 較少取代的 C=C
形成較快，較不穩定
⚖️ 熱力學烯醇陰離子
取走 三級 H（3° H）
→ 較多取代的 C=C
形成較慢，較穩定

如何選擇性地得到其中一種？

🏃 動力學烯醇陰離子 ⚖️ 熱力學烯醇陰離子
取代程度 較少取代（less substituted） 較多取代（more substituted）
穩定性 較不穩定 較穩定
使用的鹼 LDA（強、體積大） NaOEt 或 KOtBu
溶劑 THF（非質子） EtOH 或 tBuOH（質子）
溫度 –78°C（超低溫） 25°C（室溫）
🔑 判斷口訣
  • LDA + THF + –78°C → 動力學（Kinetic）→ 較少取代那邊
  • RO⁻ + ROH + 25°C → 熱力學（Thermodynamic）→ 較多取代那邊
原理解釋
動力學：LDA 體積大，優先攻擊位阻較小的（2° H），溫度低防止進一步平衡
熱力學：在質子溶劑中，烯醇陰離子可被質子化再去質子化，最終平衡趨向最穩定（更多取代的）那種

23.5 α 碳的消旋化（Racemization）

📐 Figure 23.2 — 丙酮烯醇陰離子的幾何結構
烯醇陰離子的 α 碳是 sp² 雜化、平面結構！

• O 原子、羰基 C、和 α-C 三者共面（trigonal planar）
• 每個原子都有 p 軌域，三個 p 軌域重疊形成共軛體系
• α-C 的電子對在 p 軌域上，能與 C=O 的 π 電子共軛
⚠️ 重要結果：消旋化
當 α 碳本來是立體中心（不對稱碳），形成烯醇陰離子後：

1. α-C 變成 sp² 平面，失去立體化學資訊
2. H₂O 再質子化時，從平面兩側的機率相等
3. 結果得到消旋混合物（racemic mixture），光學活性消失
消旋化機制示意

（sp², 無手性）

若 α 碳是立體中心，接觸鹼性水溶液後光學活性會逐漸消失 — 這是消旋化的特徵！

23.7 α 碳的鹵化反應（Halogenation）

將酮/醛與 X₂（Cl₂、Br₂、I₂）在酸或鹼存在下反應，可將 α-H 換成鹵素。

鹵化通式
R-C(=O)-CH₂-H
→
X₂, H⁺ 或 ⁻OH
R-C(=O)-CH₂-X
α-鹵化產物

23.7A 酸催化鹵化（Mechanism 23.3）— 單取代

反應在醋酸（CH₃CO₂H）中進行，過程分為「酮 → 烯醇」再「烯醇 + 鹵素」兩大部分：

1
質子化：H⁺ 加到 C=O 的 O 上，形成共振穩定的碳陽離子
C=O + H⁺ → C=OH⁺
2
去質子化：H₂O 取走 α-H，形成烯醇（enol）
C=OH⁺(有α-H) + H₂O → C(OH)=C（烯醇）
3
親核攻擊 X₂：α-C 攻擊鹵素，形成新的 C–X 鍵
C(OH)=C（烯醇） + X–X → C(OH⁺)–C–X + X⁻
4
去質子化：X⁻ 取走 OH 的 H，得到最終產物
→ C(=O)-CH(X)-R
⚠️ 酸催化特點
酸催化通常只發生單取代（mono-substitution），因為加了 X 後，α-C 附近電子密度降低，不容易再形成烯醇中間體。

23.7B 鹼催化鹵化（Mechanism 23.4）— 多取代

1
去質子化：⁻OH 取走 α-H，形成烯醇陰離子
C(=O)-CH₂H + ⁻OH → [烯醇陰離子] + H₂O
2
親核攻擊 X₂：α-C 攻擊鹵素，形成新的 C–X 鍵
[烯醇陰離子] + X–X → C(=O)-CH₂-X + X⁻
⚠️ 鹼催化危險！多取代！
鹼催化時，第一個 X 加進去後，因其電負性使 α-H 更容易被去質子化，因此步驟 1-2 會重複多次，導致多個 X 取代在同一個 α 碳上。

鹵仿反應（Haloform Reaction）— 甲基酮的特殊反應

📌 特殊反應
甲基酮（R-CO-CH₃）+ 過量 X₂ + 鹼 → 發生「氧化裂解」：

R-CO-CH₃ → R-COO⁻（羧酸鹽） + HCX₃（鹵仿）
C–C 鍵斷裂！這是一個非常特殊的機制（先三鹵取代，再鹼水解裂解）。
1
三步鹵化：⁻OH 去質子，X₂ 反應，重複三次
R-CO-CH₃ → R-CO-CH₂X → R-CO-CHX₂ → R-CO-CX₃
2
鹼水解裂解：⁻OH 親核加成到 C=O，⁻CX₃ 作為離去基，形成羧酸鹽
R-CO-CX₃ + ⁻OH → R-COO⁻ + HCX₃
🧪 碘仿試驗（Iodoform Test）
用 I₂ + ⁻OH 與甲基酮反應，生成碘仿（CHI₃，淡黃色固體）沉澱 = 陽性反應。
非甲基酮不產生沉澱 = 陰性反應。
（此為早期鑑別甲基酮的方法）

23.7C α-鹵羰基化合物的後續反應

生成的 α-鹵酮可以進一步：

  • 消除（Elimination）：用 Li₂CO₃ + LiBr，生成 α,β-不飽和羰基化合物（α,β-unsaturated carbonyl）

23.8C 應用：Tamoxifen（他莫昔芬）合成

🏥 真實藥物合成
Tamoxifen 是廣泛使用的乳癌治療藥物（品牌名 Nolvadex，1970 年代上市）。

合成關鍵步驟：
酮 A + NaH（強鹼）→ 烯醇陰離子 + CH₃CH₂I → 化合物 B（高產率）

B 再經多步反應轉化為 Tamoxifen（只有 Z 式有效）。

23.9 丙二酸酯合成法（Malonic Ester Synthesis）

合成目標
以丙二酸二乙酯（diethyl malonate，CH₂(CO₂Et)₂）為起始物，合成在 α 碳有一個或兩個烷基的羧酸（carboxylic acid）。
CH₂(CO₂Et)₂ → R-CH₂-COOH（一烷基）或 R-CR'-COOH（二烷基）

背景知識：脫羧反應（Decarboxylation）

關鍵反應：β-二酸加熱失去 CO₂
β-二酸（兩個 COOH 中間只隔一個碳）加熱後，透過六員環的過渡態失去 CO₂：

HOOC-CH₂-COOH →Δ→ CH₃-COOH + CO₂ ↑

β-酮酸（β-keto acid）同樣容易脫羧，生成酮。

2

  1. 找到 COOH 的 α 碳
  2. 確認 α 碳上有哪些烷基（R、R'）
  3. 每個烷基 = 一個烷基鹵化物（R–X 或 R'–X）
  4. 其餘骨架來自丙二酸二乙酯

分子內版本：合成環狀羧酸

環形成
使用二鹵化物（dihalide）Br(CH₂)ₙBr 作為親電子試劑，進行分子內 SN2，可以合成三至六員環的羧酸。
例：1,4-二溴丁烷 + 丙二酸酯 → 環戊烷羧酸

23.10 乙醯乙酸酯合成法（Acetoacetic Ester Synthesis）

合成目標
以乙醯乙酸乙酯（ethyl acetoacetate，CH₃COCH₂CO₂Et）為起始物，合成在 α 碳有烷基的甲基酮（methyl ketone）。

CH₃COCH₂CO₂Et → R-CH₂-COCH₃（一烷基）或 R-CR'-COCH₃（二烷基）

步驟完全相同，只是起始物和終產物不同

1
試劑：NaOEt
去質子化
乙醯乙酸酯的 α-H（β-酮酯，pKₐ ≈ 10.7）非常酸，NaOEt 即可
CH₃CO-CH₂-CO₂Et + ⁻OEt → 烯醇陰離子

乙醯乙酸酯合成法的記憶口訣：「酮-酯夾中間，NaOEt 去質子，R-X 烷化，酸水解脫 CO₂，得到甲基酮！」

逆合成分析

🔄 逆推步驟（以甲基酮為目標）
  1. 確認 CH₃CO– 部分（來自乙醯乙酸酯）
  2. 找到 α 碳上的烷基 R（每個 R 來自一個 R–X）
  3. 其餘骨架來自乙醯乙酸乙酯
