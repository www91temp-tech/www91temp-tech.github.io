20.1

羰基化合物介紹

羰基（C=O，carbonyl group）是有機化學中最重要的官能基。它的碳缺電子、π 鍵容易斷裂，能參與各式各樣的反應。本章從 Chapter 20 到 24 都圍繞著羰基展開。

🔑 核心結構
羰基碳是 sp² 混成、三角平面形，所有鍵角約 120°。雙鍵由 σ 鍵 and π 鍵組成，π 鍵位於平面上下。
Figure 20.1
甲醛（CH₂=O）的靜電勢圖
靜電勢圖中，紅色/黃色區域（氧原子周圍）= 電子豐富（δ⁻），藍色區域（碳原子）= 電子缺乏（δ⁺）。這直觀地說明了羰基的極性：氧吸引電子，碳成為親電中心，是親核試劑攻擊的目標。

⚡
關鍵差異：羰基碳上有沒有離去基，決定了這個化合物走「親核取代」還是「親核加成」的路徑。這是理解本章所有反應的核心！

為什麼羰基如此活潑？

氧的電負度遠大於碳，使 C=O 鍵高度極化：C 端（δ⁺）= 親電中心，O 端（δ⁻）= 電子豐富。
加上羰基碳是 sp² 混成（不擁擠），親核試劑可以容易地從上下兩面進攻 π 鍵。

記憶口訣

羰基碳「三缺一」：
① 缺電子（電負度差異）
② 不擁擠（sp² 三角形）
③ π 鍵弱（容易斷裂）
→ 所以親核試劑最愛攻擊它！

20.2

羰基化合物的兩大基本反應

記住這三個基本原則，就能看懂本章所有反應：

三大反應原則
  1. 親核試劑攻擊親電試劑
  2. π 鍵容易斷裂
  3. 與好的離去基相連的鍵容易斷裂

路徑一：親核加成（醛、酮）

醛和酮沒有離去基，所以親核試劑攻擊後，只能「留下來」——形成加成產物。

⚙️ 機制 20.1：親核加成（兩步驟）
1
親核攻擊 Nucleophilic Attack
親核試劑（Nu⁻）攻擊缺電子的羰基碳，π 鍵斷裂，電子對轉移到氧上，形成帶負電的烷氧基中間體（alkoxide）。碳從 sp² 變為 sp³。
Nu⁻ + R₂C=O → R₂C(O⁻)(Nu)
2
質子化 Protonation
水（H₂O）或其他質子來源使帶負電的氧原子質子化，得到最終的醇類加成產物。
R₂C(O⁻)(Nu) + H₂O → R₂C(OH)(Nu) + OH⁻
★ 反應性：醛 > 酮，醛空間較小（只有一個 R 基），且烷基少→電子給予少→羰基碳更缺電子→更活潑！

路徑二：親核取代（含離去基的羰基化合物）

羧酸衍生物（酸氯化物、酯、醯胺等）的羰基碳上有離去基（Z），親核取代後，離去基帶著電子對離開。

⚙️ 機制 20.2：親核取代（兩步驟）
1
親核攻擊 → 四面體中間體
Nu⁻ 攻擊羰基碳，π 鍵斷裂，形成 sp³ 的四面體中間體，氧帶負電。
2
離去基脫離 → 恢復 C=O
氧上的電子對重新形成 π 鍵，同時離去基 Z⁻ 帶著電子對離開。結果：Nu 取代了 Z。
R-CO-Z + Nu⁻ → R-CO-Nu + Z⁻

離去基能力決定反應活性順序

化合物 離去基 離去能力 反應活性
酸氯化物 RCOCl Cl⁻ 最強 最高
羧酸 RCOOH OH⁻ 中等 中等
酯 RCOOR' OR'⁻ 中等 中等
醯胺 RCONH₂ ⁻NH₂ 最弱 最低
💡 記憶方法
鹼性越強的離去基，離去能力越弱。
Cl⁻（很弱的鹼）→ 最好的離去基 → 酸氯化物反應性最高
NH₂⁻（很強的鹼）→ 最差的離去基 → 醯胺反應性最低

選擇性還原的策略

• NaBH₄，CH₃OH → 只還原 C=O → 得到烯丙醇（allylic alcohol）
• H₂（1當量），Pd/C → 只還原 C=C → 得到酮
• H₂（過量），Pd/C → 還原兩個 π 鍵 → 得到飽和醇

20.5

羰基還原的立體化學

還原反應將 sp² 混成的平面羰基碳轉變為 sp³ 四面體碳。當反應產生新的立體中心時，會發生什麼？

核心原理
羰基碳是 sp² 平面的，親核試劑（H:⁻）可以從平面的上方或下方以相等的機率攻擊 π 鍵。因此，兩種鏡像異構物以等量形成 → 得到外消旋混合物（racemic mixture）。
⚙️ 為何得到外消旋混合物？
↓
從上方攻擊
H:⁻ 從 π 鍵上方進入 → 形成某一種構型的烷氧基中間體 → 質子化後得到（S）醇
↑
從下方攻擊
H:⁻ 從 π 鍵下方進入 → 形成另一種構型的烷氧基中間體 → 質子化後得到（R）醇
★ 結論：以 NaBH₄ 或 LiAlH₄ 還原非對掌的酮時，如果產生新的立體中心，必然得到等量的兩種鏡像異構物（外消旋混合物）。
20.6

對掌選擇性羰基還原

若需要只得到單一鏡像異構物，必須使用對掌性還原試劑。這與 Sharpless 不對稱環氧化策略相同。

CBS 試劑

CBS 試劑由硼烷（BH₃） 與對掌性的噁唑硼烷（oxazaborolidine）反應製備。因由 Corey、Bakshi、Shibata 三位化學家開發，故稱 CBS 試劑。

試劑 H:⁻ 進攻方向 主要產物
(S)-CBS 試劑 從 C=O 正面進攻 R 醇（主要產物）
(R)-CBS 試劑 從 C=O 背面進攻 S 醇（主要產物）
Figure 20.3
對掌選擇性還原——沙美特羅（Salmeterol）合成的關鍵步驟
沙美特羅是一種長效支氣管擴張劑，用於治療氣喘。其合成中利用 (R)-CBS 試劑對酮進行對掌選擇性還原，得到 (R) 構型的醇 A，再經四步驟合成 (R)-沙美特羅（商品名 Serevent）。

使用 (S)-CBS 還原丙苯酮，可得到 97% ee（異構物過量）的 R 醇——幾乎只有一種異構物！

生物體內的對掌選擇性還原

細胞內的還原試劑是 NADH（菸鹼醯胺腺嘌呤二核苷酸），由維生素菸鹼酸（Niacin，大豆等食物中含有）合成。

生物體的完美選擇性
NADH 也是提供 H:⁻ 的親核試劑。與實驗室試劑不同，生物還原在酶的催化下，永遠只形成單一鏡像異構物（100% 選擇性）。

例如：丙酮酸（pyruvic acid）在乳酸去氫酶（lactate dehydrogenase）催化下，由 NADH 還原只得到 (S)-乳酸。劇烈運動時肌肉中乳酸堆積，就是因為氧氣不足，無法繼續代謝丙酮酸，只好先還原為乳酸。
20.7

羧酸衍生物的還原

羧酸衍生物（RCOZ）的還原比較複雜，因為產物取決於離去基的種類和還原試劑的強度。

三種金屬氫化試劑

LiAlH₄（強，非選擇性）
DIBAL-H（溫和）
LiAlH[OC(CH₃)₃]₃（溫和）

DIBAL-H（二異丁基氫化鋁）：兩個龐大的異丁基使它比 LiAlH₄ 活性低。

LiAlH[OC(CH₃)₃]₃：三個吸電子的氧原子降低了鋁的親核性。

酸氯化物和酯的還原

RCOCl 或 RCOOR' ‒‒ 強試劑 LiAlH₄ → RCH₂OH（1° 醇）
RCOCl 或 RCOOR' ‒‒ 溫和試劑 DIBAL-H → RCHO（醛）
Figure 20.4
DIBAL-H 在西加毒素 CTX3C 合成中的應用
西加毒素（Ciguatoxin CTX3C）是存在於400多種熱帶暖水魚中的強效神經毒素，每年有數千人因食用含此毒素的珊瑚礁魚類而中毒。2001年，研究人員完成了其實驗室合成，其中一個關鍵步驟就是：利用 DIBAL-H 將複雜多環醚中間體的酯基選擇性還原為醛基，再經後續多步驟得到目標分子。這展示了選擇性還原在複雜天然物合成中的重要性。

羧酸和醯胺的還原

重要規則
• 羧酸（RCOOH）+ LiAlH₄ → 1° 醇（唯一可用的還原反應）
• 醯胺（RCONH₂）+ LiAlH₄ → 胺（amine）⚠️（注意！不是醇）
醯胺還原為胺的原因：⁻NH₂ 是非常差的離去基，從不在還原中離開，因此氮原子保留在產物中。

20.8

醛的氧化

羰基化合物最常見的氧化反應是醛氧化為羧酸。酮因羰基碳上無氫原子，不發生此反應。

R–CHO（醛） ‒‒ CrO₃, K₂Cr₂O₇, KMnO₄ → R–COOH（羧酸）

托倫斯試劑（Tollens' Reagent）

使用 Ag₂O 在 NH₄OH 水溶液中（稱為托倫斯試劑），可選擇性氧化醛，即使其他官能基存在也不受影響。最大特色是 Ag⁺ 被還原為金屬銀（Ag），在玻璃瓶內壁形成銀鏡！

🔬 托倫斯試驗（銀鏡反應）
• 正反應（醛存在）：醛 + Ag⁺ → 羧酸 + Ag（銀鏡）
• 負反應（無醛）：無銀鏡生成

這是鑑別醛（正）和酮（負）的經典方法！

這就是為什麼反應必須在無水條件下進行。水要在加成完成後才能加入（用於質子化）。
作為親核試劑：攻擊親電的碳原子，形成新的 C–C 鍵！
20.10

有機金屬試劑與醛 / 酮的反應

這是有機合成中最重要的 C–C 成鍵反應之一！有機鋰或 Grignard 試劑與醛 / 酮反應後加水，得到醇類。

R₂C=O ‒‒ [1] R"MgX 或 R"Li
[2] H₂O → 醇（含新 C–C 鍵）

機制 20.6：親核加成

⚙️ Grignard 試劑加成到醛 / 酮
1
R"⁻ 攻擊羰基碳 → 形成新 C–C 鍵
有機金屬試劑中的碳（δ⁻）作為親核試劑，攻擊羰基碳（δ⁺），π 鍵斷裂，形成帶負電的烷氧基中間體，同時建立一個新的 C–C 鍵。
2
H₂O 質子化烷氧基 → 得到醇
加入水後，烷氧基的氧原子被質子化，形成含新 OH 基的醇類產物。

醛 / 酮種類決定醇的級別

甲醛 CH₂=O + R"–M 1° 醇 R"CH₂OH
其他醛 RCHO + R"–M 2° 醇 RCH(OH)R"
酮 R₂C=O + R"–M 3° 醇 R₂C(OH)R"
Figure 20.5
乙炔雌二醇（Ethynylestradiol）的合成
乙炔雌二醇是口服避孕藥的重要成分。其合成的關鍵步驟是：以乙炔化鋰（鋰乙炔化物）與甾體酮類進行親核加成，形成含炔基的三級醇，再一步轉化為乙炔雌二醇。這展示了有機金屬試劑在複雜天然物合成中建立 C–C 鍵的關鍵作用。

立體化學：加成也得外消旋混合物

與還原反應一樣，Grignard 加成從 sp² 平面的兩面均等進攻，若產生新立體中心，得到外消旋混合物（等量的兩種異構物）。

20.11

Grignard 產物的逆合成分析

逆合成分析（Retrosynthetic Analysis）是從目標分子「倒推」所需原料的思維策略，是有機合成設計的核心工具。

逆合成的箭頭符號
使用雙線箭頭 ⟹ 表示「反推」（與合成方向相反）
找到羰基碳（未來的 C–OH 碳）兩側可能斷開的 C–C 鍵

2° 醇的逆合成

2° 醇通常有兩種逆合成路線（以丁烷-2-醇為例）：

路線一：
butan-2-ol ⟹ propanal + CH₃MgX
路線二：
butan-2-ol ⟹ acetaldehyde (CH₃CHO) + CH₃CH₂MgX
逆合成步驟口訣

目標醇 C–OH 的碳是原本的羰基碳：

  • 確認 C–OH 兩側各有哪些基團
  • 選擇從哪個 C–C 鍵斷開
  • 斷開處：一側給 Grignard（R"MgX），另一側給醛 / 酮
  • 3° 醇有三種路線（三個不同 C–C 鍵可斷開）
20.12

保護基

有個大問題：Grignard 試劑是強鹼，如果分子中同時含有OH 或 NH 基團，Grignard 試劑會先奪取質子，而不是進行親核加成！

⚠️
含有 ROH、RCOOH、RNH₂、R₂NH、RCONH₂、RSH 的化合物，會與有機金屬試劑發生酸鹼反應，而不是親核加成！

保護基策略（三步法）

Figure 20.7
保護基的一般策略
以 5-羥基戊烷-2-酮（含 OH 和 C=O 兩個官能基）加 CH₃MgCl 為例：

步驟一 保護（Protection）：將 OH 轉化為 TBDMS 醚（叔丁基二甲基矽基醚），加入 TBDMS-Cl + 咪唑
步驟二 反應：加入 CH₃MgCl，進行親核加成，再加水質子化，得到受保護的三級醇
步驟三 去保護（Deprotection）：加入四丁基氟化銨（Bu₄N⁺F⁻），F⁻ 親矽性強，切斷 O–Si 鍵，恢復 OH 基

最終產物：4-甲基戊烷-1,4-二醇

TBDMS 矽醚保護基

🛡️ 保護和去保護
保護：R–OH + TBDMS-Cl + 咪唑 → R–O–Si(CH₃)₂C(CH₃)₃（矽醚）
去保護：R–O–TBDMS + Bu₄N⁺F⁻ → R–OH + F–Si（矽醚斷裂）

F⁻ 對矽有很強的親和力（Si–F 鍵極強），因此用氟化物可選擇性切除矽醚保護基。
20.13

有機金屬試劑與羧酸衍生物反應

有機鋰和 Grignard 試劑與酯或酸氯化物反應，加兩當量試劑，得到三級醇（兩個相同的 R 基）。

RCOCl 或 RCOOR' ‒‒ [1] 2×R"MgX
[2] H₂O → R–C(OH)(R")₂（3° 醇）

有機銅（Cuprate）與酸氯化物：製備酮

要止步於酮（不繼續到 3° 醇），需要用反應性較弱的有機銅試劑，且只有酸氯化物（有最佳離去基 Cl⁻）能反應：

RCOCl ‒‒ R'₂CuLi → R–CO–R'（酮）
20.14

Grignard 與 CO₂、環氧化物反應

Grignard 試劑與 CO₂ — 羧化反應（carboxylation）
R–MgX + CO₂ → 烷氧基中間體 →（H₃O⁺ 質子化）→ R–COOH（羧酸）

這讓鹵烷可以轉化為多一個碳的羧酸（常用於增碳）：
R–X → R–MgX（加 Mg / 乙醚）→ R–COOH（加 CO₂，再酸化）
Grignard 試劑開環氧化物
有機金屬試劑（RLi、RMgX、R₂CuLi）都能開環氧化物的環，形成醇。
規則：親核攻擊發生在空間較小（位阻較小）的碳上，且為 SN2 背面進攻。
20.15

α,β-不飽和羰基化合物

α,β-不飽和羰基化合物是含有共軛的 C=C 和 C=O 兩個π鍵的分子（中間只隔一個σ鍵）。

共振結構的重要性
三個共振結構顯示：除了羰基碳（δ⁺），β 碳也帶有部分正電（δ⁺）！
→ 親核試劑可以攻擊兩個不同位置：羰基碳（C=O 端）或 β 碳

🏆 期末考前必記要點
  • 羰基碳 = 親電中心（δ⁺），永遠被親核試劑攻擊
  • 醛比酮活潑（空間小 + 電子密度低）
  • NaBH₄ 選擇性還原 C=O；H₂/Pd 先還 C=C
  • LiAlH₄ 還原醯胺得胺（不是醇！）
  • Grignard/有機鋰是強親核試劑+強鹼，必須無水操作
  • 有機銅（cuprate）活性弱，用於製備酮或 1,4-加成
  • 含 OH/NH 的底物加 Grignard 前，先用保護基保護
  • 對掌選擇性還原用 CBS 試劑；生物體用 NADH
  • 托倫斯試驗（銀鏡反應）只有醛呈陽性，酮呈陰性
