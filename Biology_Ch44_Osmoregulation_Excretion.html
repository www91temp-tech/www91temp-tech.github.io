滲透調節與排泄系統

Osmoregulation and Excretion

動物如何在體內維持水分與鹽分的平衡？廢物又如何從體內排出？

🌊 核心問題：動物如何調節體液中的水分與鹽分濃度？
📖 前言 💧 滲透調節 🧬 含氮廢物 🔬 排泄系統 🫘 腎元運作 ⚗️ 荷爾蒙調控 📝 複習測驗

🎯 為什麼需要滲透調節和排泄？

動物的細胞必須浸泡在濃度適當的體液中才能正常運作。如果體液太稀或太濃，細胞會膨脹破裂或萎縮死亡。同時，細胞代謝蛋白質和核酸時會產生有毒的含氮廢物（如氨），必須及時排出體外。

🫧

高滲（Hyperosmotic）

細胞外比細胞內
溶質濃度更高
→ 水分流出細胞
→ 細胞皺縮死亡 💀

⚖️

等滲（Isoosmotic）

兩側溶質濃度相同
→ 水分不淨移動
→ 細胞保持正常 ✅

🌵

低滲（Hypoosmotic）

細胞外比細胞內
溶質濃度更低
→ 水分進入細胞
→ 細胞膨脹破裂 💥

⚠️ 關鍵概念 動物無法直接抽水穿過細胞膜，所以只能藉由移動溶質來間接控制水分的流向。水分會從溶質濃度低的地方，透過滲透作用（osmosis）流向溶質濃度高的地方。
44.1

💧滲透調節：平衡水分與溶質的進出

📏 滲透濃度（Osmolarity）是什麼？

衡量溶液濃度的單位是「滲透濃度」，單位為 mOsm/L（毫滲透莫耳/公升）。

  • 人體血液：約 300 mOsm/L
  • 海水：約 1,000 mOsm/L（是血液的3倍多）
  • 淡水：約 0.5–15 mOsm/L（遠比血液稀）

兩種應對策略

🌊

順滲調節者
Osmoconformer

體液滲透濃度與環境相同，不主動調節。幾乎只有海洋無脊椎動物（如海蛤、海星）採用此策略。省能量，但無法離開穩定的海洋環境。

🐟

滲透調節者
Osmoregulator

主動調節體液滲透濃度，使其不隨外界變化。所有淡水動物、陸生動物，以及大多數海洋脊椎動物都是此類型。需消耗能量，但適應力強。

廣鹽性 euryhaline：能承受大幅鹽度變化的動物（如鮭魚）
窄鹽性 stenohaline：對鹽度變化敏感，無法耐受大幅波動

不同棲地動物的挑戰與應對

🐠

海洋硬骨魚（如鱈魚）

海水比體液更濃，魚體不斷失水。
應對：大量喝海水 + 由鰓和腎臟排出多餘的鹽 + 排少量尿液。

🦈

軟骨魚（鯊魚）

體內保留高濃度的尿素和 TMAO（氧化三甲胺），使體液滲透濃度略高於海水。
水分因此緩慢流入，不需喝海水。

🐟

淡水魚（如鱸魚）

淡水比體液更稀，水分不斷滲入。
應對：排大量稀尿（不喝水）+ 鰓主動攝取鹽分。

🦎

陸生動物

最大威脅是脫水（desiccation）。
應對：防水皮膚（角質層、蠟質）＋夜行習性＋排出濃縮尿液＋補充飲水與食物中的水分。

🐦 有趣案例：信天翁喝海水也能活！ 海鳥（信天翁）有鼻腺（鹽腺），可分泌濃度遠高於海水的鹽溶液，直接從鼻尖滴出！喝進去的海水帶來大量鹽分，但鹽腺能排出更多的鹽，讓身體反而獲得淨水分。人類喝海水只會更脫水，因為我們的腎臟需要用更多水來排出鹽分。
🧠 記憶口訣

「海魚喝海水排少尿，淡魚不喝排大量（稀尿），陸生防水靠皮膚」——三種環境，三種策略，方向完全相反！

44.2

🧬含氮廢物的種類與演化

為什麼動物需要排出含氮廢物？

細胞分解蛋白質和核酸時，會切除胺基（–NH₂），形成具有毒性的氨（Ammonia, NH₃）。不同動物將氨轉換成不同形式的廢物排出，以平衡毒性、耗能、耗水三者之間的取捨。

🐠

氨（Ammonia）

毒性：高
耗能：低
耗水：多

直接排放，溶於大量水中即可稀釋。
主要：水生動物（魚、水生無脊椎動物）

🐭

尿素（Urea）

毒性：中
耗能：中
耗水：中

在肝臟由氨合成，溶於尿液排出。
主要：哺乳類、大部分兩棲類、鯊魚

🦅

尿酸（Uric Acid）

毒性：低
耗能：高
耗水：少

幾乎不溶於水，以半固體糊狀排出（鳥糞白色部分！）
主要：鳥類、爬行類、昆蟲、陸生蝸牛

⭐ 最重要的比較表格
比較項目 氨 Ammonia 尿素 Urea 尿酸 Uric Acid
毒性 高 低 極低
排出耗水量 多（需大量稀釋） 中 少（半固體）
合成耗能 低（不需轉換） 中 高（最耗 ATP）
溶解性 高 高 低（幾乎不溶）
代表動物 大多數魚類、水生無脊椎 哺乳類、鯊魚、成年兩棲類 鳥類、爬行類、昆蟲
🥚 演化的智慧：有殼的蛋為什麼選尿酸？ 鳥類和爬行類的蛋有硬殼，不透液體。如果胚胎排出可溶的氨或尿素，廢物會積在蛋內達到中毒濃度。尿酸不溶於水，以固體形式儲在蛋裡無害，孵化後留在殼中——這是自然選擇的精妙設計！
🧠 記憶口訣

「魚氨鳥酸哺乳尿素」——水生魚類用氨（水多稀釋沒問題）、鳥類爬行類用尿酸（省水省重要）、哺乳類用尿素（折中方案）。兩棲類幼體（蝌蚪）在水中→排氨；成體（青蛙）上陸→改排尿素。

🔬各種排泄系統——管狀構造的變奏

排泄系統的四個共同步驟

🩸
步驟 1
過濾
Filtration
血壓將水分與小分子溶質
壓入管腔，形成「原始濾液」。大分子（蛋白質）和細胞留在血液中。
♻️
步驟 2
再吸收
Reabsorption
有用的物質（葡萄糖、胺基酸、水分、鹽）從濾液回收入血液。
💉
步驟 3
分泌
Secretion
多餘的離子、藥物、毒素等從血液主動分泌進入濾液。
🚽
步驟 4
排泄
Excretion
最終的廢液（尿液）含有含氮廢物，排出體外。

不同動物的排泄系統比較

動物 排泄器官 特點 主要功能
扁蟲 🪱
（渦蟲）		 原腎管
Protonephridia		 末端有「焰球」（flame bulb），纖毛擺動帶動液體流動，不直接接觸血液 主要是滲透調節（淡水種），排出稀尿以平衡滲透進入的水

🧠 記憶口訣

「扁蟲原腎、蚯蚓後腎、昆蟲馬氏、脊椎腎臟」
複雜度依序增加，功能也越來越精密。昆蟲的馬氏管最特別——廢物先進腸道，水分在直腸最後才被回收，極度省水！

44.4

🫘腎元：濾液的逐步加工廠

腎臟的基本結構

人類每個腎臟約 10 公分，含約 100萬條腎元。腎臟分為外層的腎皮質和內層的腎髓質。腎元類型：

  • 皮質腎元 85%，位於皮質，作一般過濾
  • 近髓質腎元 15%，深入髓質，是濃縮尿液的關鍵！
⭐ 必讀：腎元各部位的功能
— 濾液在腎元中的旅程 —
①

鮑氏囊 + 腎絲球（Bowman's Capsule + Glomerulus）

血壓將血液中的水分和小分子「壓」入鮑氏囊，形成原始濾液。每日過濾約 180 公升！（大分子蛋白質和血球留在血液中）

②

近曲小管（Proximal Tubule）

大量再吸收：NaCl、水（約65%）、葡萄糖、胺基酸全部回收。同時分泌 H⁺ 調節 pH 值。藥物和毒素在此由血液分泌進入濾液排出。

③

亨氏環下行支（Descending Limb of Loop of Henle）

只透水，不透鹽——水分依滲透梯度大量離開（有水通道蛋白 aquaporin），濾液越走越濃縮，在彎底部滲透濃度最高（約 1,200 mOsm/L）。

④

亨氏環上行支（Ascending Limb of Loop of Henle）

只透鹽，不透水——NaCl 擴散或主動運輸離開（無水通道蛋白），濾液越走越稀，回到皮質時已比血液還稀。上行支主動運輸鹽是維持腎髓質高滲梯度的核心機制！

⑤

遠曲小管（Distal Tubule）

調節 K⁺ 和 NaCl 的濃度；受荷爾蒙調控，協助維持體液 pH 值（分泌 H⁺，再吸收 HCO₃⁻）。

⑥

集尿管（Collecting Duct）

決定最終尿液濃度的關鍵！

• 有 ADH → 水通道蛋白插入膜中 → 大量吸水 → 排出濃縮尿液

• 無 ADH → 無水通道蛋白 → 主動排鹽 → 排出大量稀尿

• 內髓質段透過尿素 → 尿素擴散出去，維持髓質高滲梯度

逆流倍增系統——濃縮尿液的秘密

🔄 逆流倍增系統（Countercurrent Multiplier）

亨氏環的下行支和上行支的液體方向相反，就像兩條方向相反的河流並排而流，形成一個不斷「放大」滲透梯度的機制：

  • 上行支主動運輸鹽分出去（消耗 ATP），使腎髓質越深越鹹
  • 下行支水分被抽走，濃縮的鹽又流進上行支的底端，繼續循環
  • 最終在腎髓質深部建立高達 1,200 mOsm/L 的超高滲透環境
  • 集尿管流過這個梯度時，水分自然被抽出，尿液因此濃縮
⚠️ 亨氏環長度決定尿液濃縮能力

亨氏環越長→ 滲透梯度越大 → 尿液越濃縮（越省水）

  • 沙漠哺乳類（如袋鼠鼠）：亨氏環極長，尿液可濃縮至 9,300 mOsm/L（是血液的31倍！）
  • 水生哺乳類（如海狸）：多為短亨氏環的皮質腎元，尿液較稀
  • 沙漠動物不喝水也能生存的關鍵就在這裡！
🧠 記憶要點

記住亨氏環兩支的「天生對立」：下行支透水不透鹽；上行支透鹽不透水。這個不對稱性是逆流倍增的核心！人類每天過濾 180 公升的血液，但最終只排出約 1.5 公升尿液——99% 的水分被回收，這就是腎元的驚人效率。

44.5

⚗️荷爾蒙調控——腎臟、水分與血壓的連動

腎臟受三套荷爾蒙系統的精密調控

腎臟並不是只管自己的事——它的運作與血壓、血量、體液滲透濃度全部相互連動，由以下三套系統共同協調：

💧 ADH（抗利尿激素）——調控水分保留

完整名稱：Antidiuretic Hormone，又叫 vasopressin（血管加壓素）

觸發條件：血液滲透濃度升高（例如：流汗後、攝入過多鹽分）

釋放位置：下視丘偵測 → 腦垂體後葉釋放 ADH

作用機制：ADH 與集尿管細胞受體結合 → 啟動訊號傳遞 → 水通道蛋白（aquaporin）被插入細胞膜 → 管壁對水的通透性大增 → 水分大量被吸回 → 排出濃縮小量尿液

🍺 補充：酒精會抑制 ADH 分泌 → 水通道蛋白減少 → 大量排尿 → 脫水，這就是宿醉頭痛的部分原因！
💉 尿崩症（Diabetes Insipidus）：ADH 缺乏或受體失效 → 集尿管無法插入水通道蛋白 → 24小時大量排出極稀的尿液

⚡ RAAS 系統——調控血壓與血量

完整名稱：Renin-Angiotensin-Aldosterone System（腎素–血管緊張素–醛固酮系統）

觸發條件：血壓或血量下降（如大量失血、嚴重腹瀉）——注意：此時滲透濃度不一定改變，ADH不會啟動，所以需要另一套系統！

作用流程：

  1. 腎絲球旁器（JGA）偵測到進入腎絲球的動脈壓力下降
  2. JGA 釋放腎素（Renin）酵素
  3. 腎素切割肝臟分泌的血管緊張素原 → 最終生成血管緊張素II
  4. 血管緊張素II 做兩件事：①使小動脈收縮→血壓上升；②刺激腎上腺皮質分泌醛固酮（Aldosterone）
  5. 醛固酮促使遠曲小管和集尿管多吸收 Na⁺ 和水 → 血量增加 → 血壓回升
💊 臨床應用：許多高血壓藥物（ACE抑制劑）正是藉由阻斷血管緊張素II的生成來降低血壓！

🌿 ANP——對抗 RAAS 的制衡機制

完整名稱：Atrial Natriuretic Peptide（心房鈉尿肽）

觸發條件：血量和血壓上升時，心房壁受到牽張，釋放 ANP

效果：抑制腎素釋放 + 抑制集尿管吸收 NaCl + 減少醛固酮分泌 → 降低血量和血壓

⭐ 三套系統的協調總覽
系統 觸發條件 效果
ADH 血液滲透濃度↑（脫水、吃鹹） 水通道蛋白↑ → 尿液減少、濃縮
RAAS 血壓/血量↓（失血、嘔吐腹瀉） Na⁺ 和水的再吸收↑ → 血壓/血量↑
ANP 血壓/血量↑（體液過多） 對抗 RAAS → 鈉和水排出↑ → 血壓↓
🧠 記憶口訣

「血濃靠ADH保水；血少靠RAAS升壓；血多靠ANP洩洪」
三套系統互相制衡，像一個精密的「水壓三角」，共同維持身體的液體平衡！

快速複習測驗——點擊題目查看答案

🎯 核心概念自我測驗
海洋硬骨魚（如鱈魚）面臨什麼滲透調節問題？牠們如何解決？
海水滲透濃度（約1,000 mOsm/L）遠高於體液（約300 mOsm/L），魚體不斷被海水「吸走」水分（低滲環境）。解決方式：大量喝海水補充水分，然後由鰓主動運輸排出多餘的鹽分（NaCl），並排出少量濃縮的尿液。
為什麼淡水魚不需要喝水，卻需要排大量的稀尿？
淡水滲透濃度極低（約0.5–15 mOsm/L），而淡水魚體液約300 mOsm/L，水分會不斷滲入魚體（魚是高滲境中，水往裡跑）。魚必須排出大量稀尿來排除多餘水分，同時鰓需主動攝取鹽分補充因尿液流失的離子。不需喝水，因為水自動流進來！
比較氨、尿素、尿酸三者，各有什麼優缺點？
氨（Ammonia）：直接排出，合成能耗最低，但毒性高，需大量水稀釋，只適合水生動物。
尿素（Urea）：毒性低、溶於水，排出耗水量中等；但需在肝臟合成，耗費一些能量。哺乳類、成年兩棲類採用。
尿酸（Uric Acid）：幾乎不溶，可以半固體排出，耗水最少（最適合缺水環境）；但合成耗能最高。鳥類、爬行類、昆蟲採用。
亨氏環的下行支和上行支各有什麼通透性特點？這對腎臟濃縮尿液有何作用？
下行支：透水（有水通道蛋白aquaporin），不透鹽 → 水分滲出，濾液越走越濃縮。
上行支：透鹽，不透水（無水通道蛋白） → 鹽分離開，濾液越走越稀。
整體效果：NaCl不斷被抽到腎髓質，形成從皮質到髓質越來越高的滲透梯度。集尿管流過此梯度時，水分被抽走，尿液越來越濃縮。這個「逆流倍增系統」是哺乳類能保水的核心機制。
喝酒後為什麼會特別頻尿？請用ADH解釋。
酒精會抑制腦垂體後葉釋放ADH。ADH減少 → 集尿管的水通道蛋白（aquaporin）插入量減少 → 管壁對水的通透性降低 → 水分無法被再吸收 → 排出大量稀薄尿液（利尿效果）→ 身體脫水。這也是宿醉頭痛（腦部輕微脫水）的原因之一。
RAAS系統在什麼情況下啟動？說明其完整作用路徑。
觸發條件：血壓或血量下降（如失血、嚴重腹瀉、大量嘔吐）。
路徑：腎絲球旁器（JGA）偵測血壓下降 → 釋放腎素（renin） → 腎素切割血管緊張素原 → 生成血管緊張素II → ①動脈收縮，血壓立即升高；②刺激腎上腺皮質分泌醛固酮（aldosterone） → 遠曲小管和集尿管增加 Na⁺ 和水的吸收 → 血量增加 → 血壓回升 → 負回饋抑制腎素繼續分泌。
為什麼沙漠哺乳類能產生極度濃縮的尿液，而水生哺乳類不行？
關鍵在於近髓質腎元的比例與亨氏環長度。沙漠哺乳類（如澳洲跳鼠）具有大量近髓質腎元，且亨氏環極深入腎髓質，能建立更陡的滲透梯度。相比之下，水生哺乳類（如海狸）多為皮質腎元，亨氏環短，滲透梯度淺，排出的尿液較稀。自然選擇依照棲地需求塑造了腎元的結構！
鳥類的蛋為什麼一定要使用尿酸作為含氮廢物排泄？
鳥蛋有防水的硬殼，氣體可透過，但液體不行。如果胚胎排出可溶的氨（毒性高）或尿素（可溶），廢物會在密閉的蛋內積累至有毒濃度，導致胚胎死亡。尿酸幾乎不溶於水，可以固體形式安全儲存在蛋內，等孵化時留在殼中，對胚胎無害。這是鳥類在有殼蛋中演化出的重要適應！
