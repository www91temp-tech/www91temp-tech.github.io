導論：認識醛與酮

§21.1

什麼是羰基化合物？

醛與酮都含有 C=O（羰基，carbonyl group）。

  • 醛（RCHO）：羰基碳上至少連接一個 H 原子
  • 酮（RCOR'）：羰基碳上連接兩個烷基或芳基（沒有 H）

⚠ 醛的寫法 RCHO 中，H 是接在 C 上，不是 O 上！

羰基碳的兩大特性：
① sp² 混成，平面三角形（鍵角約 120°，空間較不擁擠）
② 氧原子電負度大，使羰基碳帶 δ⁺（親電子性） → 容易被親核試劑攻擊

反應活性大小

羰基碳上的烷基越多，活性越低（立體效應＋電子效應）：

圖示中，箭頭向左代表「對親核試劑的反應活性增加」，從酮(R₂C=O)到醛(RCHO)再到甲醛(H₂C=O)；箭頭向右代表「立體障礙增加」。烷基取代越多，阻擋親核試劑接近的空間越大，且供電子烷基使羰基碳δ⁺降低，故活性越低。

本章開頭介紹的天然藥物 digoxin（地高辛） 用於治療充血性心臟衰竭，含有三個縮醛單元（acetals），這些縮醛正是由羰基加成反應形成的。每 1000 kg 乾燥毛地黃葉可萃取 1 kg 的 digoxin。

命名規則

§21.2

醛的 IUPAC 命名

兩種情況

CHO 接在鏈上 → 找含 CHO 的最長碳鏈，把母烷烴的 -e 改成 -al；CHO 碳在 C1（省略數字 1）

CHO 接在環上 → 環名 + -carbaldehyde，環的 CHO 附著點為 C1。

範例結構IUPAC 名稱說明
CH₃CHOethanal（乙醛）2C，後綴 -al
CH₃CH₂CHOpropanal（丙醛）3C，CHO 在 C1
2,3-二甲基丁醛2,3-dimethylbutanal最長鏈4C，甲基在C2、C3
環己烷-CHO（乙基在C2）2-ethylcyclohexanecarbaldehyde環上用 -carbaldehyde
醛俗名口訣：把羧酸俗名後綴 –ic acid 改為 –aldehyde。

常用俗名（必背！）

formaldehyde 甲醛
IUPAC: methanal
工業重要原料，防腐劑（福馬林 formalin = 37% 水溶液）
acetaldehyde 乙醛
IUPAC: ethanal
乙醇氧化的中間產物
benzaldehyde 苯甲醛
IUPAC: benzenecarbaldehyde
苦杏仁味，用於食品香料

α、β 位置標示（俗名系統）

俗名系統中，緊鄰 CHO 的碳叫 α碳，再下一個是 β碳，依此類推。

CHO — Cα — Cβ — Cγ — ...

⚠️ IUPAC 的編號從 C=O 碳（C1）開始往外數；希臘字母編號從緊鄰 C=O 的碳（α）開始。

酮的 IUPAC 命名

找含 C=O 的最長碳鏈，後綴改 -one，給 C=O 碳最小編號。

環狀酮：編號從羰基碳（C1）開始，取代基取得最小號碼方向。

Figure 21.2 說明
兩個酮的命名範例

① 2-methylpentan-3-one：5碳鏈，C3 為羰基碳，C2 有甲基；俗名 ethyl isopropyl ketone
② m-bromoacetophenone（3-bromoacetophenone）：苯環上的間位（C3）有 Br，C1 連接乙醯基

⚗️ acetone 丙酮（propan-2-one）最常用有機溶劑；糖尿病患者血液中偏高

物理與光譜性質

§21.3–21.4

物理性質

醛與酮的極性羰基使分子間有偶極–偶極作用力，但沒有 O−H 鍵，所以無法互相形成氫鍵。

  • 沸點比分子量相近的醇低，但比烴類高
  • ≤5 碳的醛酮與水互溶（可與水形成氫鍵）；>5 碳者不溶於水

反應通則：親核加成機制

§21.7

醛與酮的特徵反應是親核加成（Nucleophilic Addition）：
親核試劑 Nu:⁻ 攻擊帶 δ⁺ 的羰基碳 → C–Nu 鍵形成，C=O π鍵斷裂 → 氧負離子 → 質子化。
總結：H 和 Nu 加到 C=O 上。

機制 21.1：帶負電荷親核試劑（強親核試劑）

Mechanism 21.1 — 兩步驟（親核攻擊先、質子化後）
1
親核試劑 Nu:⁻ 攻擊羰基碳
C=O 的 π 鍵斷裂，電子推到 O 上，形成帶負電的烷氧基陰離子（alkoxide）
C 從 sp² 變成 sp³（四面體）
2
O⁻ 被 H₂O 質子化 → 形成加成產物（含 OH 基）
同時再生 ⁻OH

機制 21.2：中性弱親核試劑需要酸催化

Mechanism 21.2 — 三步驟（質子化先、親核攻擊後）
1
酸 H–A 先將羰基氧質子化 → 形成共振穩定的碳陽離子（C=OH⁺）
羰基碳帶淨正電荷，親電子性大增
2
弱親核試劑 H–Nu: 攻擊活化後的羰基碳
3
去質子化 → 中性加成產物；酸催化劑 H–A 再生

酸催化的意義

中性羰基：電中性，弱親核試劑難以攻擊
質子化羰基（C=OH⁺）：帶淨正電荷，親電子性大幅增加
→ 凡是遇到強酸，第一步永遠是「羰基氧先被質子化」！

Cl⁻、Br⁻、I⁻ 雖是好的取代反應親核試劑，但不有效加成到羰基上——因為它們是比生成的烷氧基更弱的鹼，平衡偏向起始物。

本章親核加成反應總覽

親核試劑產物催化章節
NaBH₄ 或 LiAlH₄（H:⁻）醇（alcohol）無需§21.8
RMgX 或 RLi（R:⁻）醇（alcohol，含新 C–C 鍵）無需§21.8
⁻CN（NaCN/HCl）氰醇（cyanohydrin）無需§21.9
Ph₃P=CR₂（Wittig 試劑）烯烴（alkene）無需§21.10

R₂NH（2° 胺）烯胺（enamine，C=C–N）弱酸§21.12
H₂O偕二醇（gem-diol）酸或鹼§21.13
ROH（醇，2 當量）縮醛（acetal）酸§21.14

氫化物與碳陰離子加成（複習）

§21.8
還原到醇
RCHO 或 R₂C=O ── NaBH₄/CH₃OH 或 [1] LiAlH₄ [2] H₂O ──→ 1° 或 2° 醇
Grignard / 有機鋰加成
RCHO 或 R₂C=O ── [1] R''MgX 或 R''Li [2] H₂O ──→ 1°/2°/3° 醇（含新 C–C 鍵）
Grignard 加成機制（兩步驟）
1
R''⁻（碳陰離子）攻擊 sp² 羰基碳，從兩側都可進攻（因為平面三角形）→ 形成烷氧基陰離子（sp³ 中間體）
2
H₂O 質子化 O⁻ → 得到醇產物

由於親核試劑從平面兩側進攻 sp² 羰基碳，若產生新的立體中心，會得到兩種非鏡像異構物（diastereomers）的混合物。

氰醇的形成（⁻CN 加成）

§21.9
氰醇形成
醛或酮 + NaCN / HCl ──→ cyanohydrin（氰醇）：−OH 和 −CN 同在一個碳上

氰醇形成的意義：增加一個碳原子，形成新的 C–C 鍵（−CN 碳接到原本羰基碳上）。

Mechanism 21.3 — 氰醇形成（兩步驟）
1
CN⁻（強親核試劑）攻擊羰基碳 → 形成新 C–C 鍵，C=O π鍵斷裂，O 帶負電
2
O⁻ 被 HCN 質子化 → 得到氰醇；同時再生 CN⁻（HCN 由 CN⁻ + HCl 生成）
單純 HCN（不含 CN⁻）無法使反應發生。必須有游離的 CN⁻ 作為親核試劑才能攻擊羰基。

氰醇的進一步轉化

水解：氰醇的 –CN 可在酸或鹼加熱下水解成 –COOH（三個 C–N 鍵全被 C–O 鍵取代），製得 α-羥基酸（α-hydroxy acid）。

逆反應：鹼處理氰醇 → 去質子化，再脫離 CN⁻ → 回到羰基化合物。

天然氰醇衍生物
• linamarin：存在於木薯（cassava）中，代謝後釋放有毒 HCN
• amygdalin：存在於桃、杏、野生櫻桃的核仁中，代謝同樣產生 HCN
• 木薯若不妥善加工（水煮或烤），其中的 linamarin 被酵素分解釋放 HCN，可導致中毒甚至死亡。
• Amygdalin 和其合成衍生物 laetrile 曾被吹捧為抗癌藥，但療效未獲證實。

Wittig 反應

§21.10

核心概念：C=O 變成 C=C

Wittig 反應用 Wittig 試劑（phosphorus ylide） 將醛或酮的羰基轉換成烯烴：

R₂C=O  +  Ph₃P⁺–⁻CR"₂  →  R₂C=CR"₂  +  Ph₃P=O

Wittig 試劑的製備（兩步驟）

合成磷葉立德（Ylide）
1
SN2 反應：三苯基膦（Ph₃P:）攻擊1° 鹵烷（最好是 CH₃X 或 RCH₂X）→ 形成磷鹽（phosphonium salt）
Ph₃P: + R"CH₂X → Ph₃P⁺–CH₂R" X⁻
2
強鹼（如 BuLi）去質子化磷鹽上的 C–H → 形成碳陰離子（ylide）
Ph₃P⁺–CH₂R" + BuLi → Ph₃P⁺–⁻CHR" ↔ Ph₃P=CHR"

Wittig 反應機制（兩步驟）

Mechanism 21.4 — Wittig 反應
1
Ylide 的帶負電碳攻擊羰基碳，同時 O 攻擊帶正電 P → 形成四員環中間體 oxaphosphetane（含 O–C–C–P 環）
2
oxaphosphetane 消去分解：形成 烯烴（新 C=C）和 Ph₃P=O（強 P=O 鍵提供反應驅動力）

Wittig 反應一次形成 兩個 C–C 新鍵（一個 σ 鍵、一個 π 鍵）。
副產物為 Ph₃P=O（三苯基氧化膦），形成強 P=O 鍵是反應自發進行的驅動力。

Wittig 反應的限制：有時會形成 E/Z 烯烴異換物的混合物（無法完全控制立體化學）。
β-胡蘿蔔素（β-carotene）的合成（Figure 21.8）
β-胡蘿蔔素是胡蘿蔔中的橙色色素，也是維生素 A 的前驅物。其合成利用 Wittig 反應一步形成中間的共軛雙鍵，是 Wittig 反應在天然物合成中的典型應用。反應中 E 式烯烴為主要產物。

逆合成分析：由烯烴推回起始物

將 C=C 斷開（切在你想形成的那個雙鍵），其中一端接 O → 這就是醛/酮；另一端接 Ph₃P= → 這就是 Wittig 試劑。

通常選擇將 Ph₃P= 接在較不擁擠的那端，以提高產率。

胺類加成：亞胺與烯胺

§21.11–21.12

1° 胺 → 亞胺（Imine / Schiff Base）

亞胺形成（弱酸催化）
醛或酮 + R''NH₂ ──弱酸(pH 4-5)──→ 亞胺（R₂C=NR''）+ H₂O

反應結果：C=O 被 C=N 取代
中間體：carbinolamine（羥胺中間體，OH 和 NHR'' 在同一碳上）→ 失去 H₂O → 亞胺

Mechanism 21.5 — 亞胺形成（五步驟）
1
1° 胺的 N（孤電子對）攻擊羰基碳 → 形成 N–C 鍵（sp³ 中間體）
2
質子轉移（proton transfer）→ 形成 carbinolamine（R₂C(OH)(NHR'')）
3
酸將 OH 質子化 → –OH₂⁺（好的離去基）
4
失去 H₂O → 形成共振穩定的 iminium ion（C=N⁺HR''）
5
失去 N 上的 H（去質子化）→ 形成亞胺（C=NR''）
最適 pH 為 4–5（弱酸）。酸性太強時：胺被完全質子化（RNH₃⁺），失去孤電子對，無法進行親核攻擊！
視覺的化學（Figure 21.9）
視網膜桿細胞中，11-cis-retinal（視黃醛）與蛋白質 opsin 中的 1° 胺發生亞胺形成反應，生成 rhodopsin（視紫素）。
光照射後：11-cis 雙鍵異構化為 11-trans → 蛋白質構形劇變 → 改變 Ca²⁺ 濃度 → 傳遞神經衝動到大腦 → 產生視覺！

2° 胺 → 烯胺（Enamine）

烯胺形成（弱酸催化）
醛或酮 + R₂NH ──弱酸──→ 烯胺（N 接在 C=C 上）+ H₂O
Figure 21.10 說明
1° 胺 vs 2° 胺與羰基化合物反應的比較

兩者機制相同，差別在最後一步：
• 1° 胺：iminium ion 中間體上 N 還有 H，可失去 N–H 質子 → 形成 C=N（亞胺）
• 2° 胺：iminium ion 中間體上 N 沒有 H，只能失去相鄰 C–H 質子 → 形成 C=C（烯胺）

亞胺 vs 烯胺的關鍵區別：
1° 胺（R–NH₂）→ imine（C=NR，氮上無多餘H時停止）
2° 胺（R₂–NH）→ enamine（C=C，因為N上無H可失去）

水解（逆反應）

亞胺和烯胺均可在稀酸（H₃O⁺）中水解，機制為形成的逆反應，分別回到原本的醛/酮與胺。

水的加成（水合反應）

§21.13
水合反應
R₂C=O + H₂O ──H⁺ 或 ⁻OH──→ gem-diol（偕二醇 / 水合物）R₂C(OH)₂

熱力學：哪種化合物水合程度高？

化合物水合物比例原因
甲醛（HCHO）99.9%最不穩定的羰基（無烷基），平衡偏向水合物
乙醛（CH₃CHO）58%一個供電子甲基，羰基稍穩定
丙酮（(CH₃)₂CO）0.2%兩個烷基穩定羰基，幾乎不水合
三氯乙醛（CCl₃CHO）接近100%三個 Cl 吸電子，大幅去穩定羰基

烷基（供電子）→ 穩定羰基 → 水合比例↓
拉電子基（如 CCl₃）→ 去穩定羰基 → 水合比例↑

動力學：酸鹼均可催化

鹼催化：⁻OH 直接作為強親核試劑攻擊羰基（機制 21.7，兩步驟）

酸催化：H⁺ 先質子化羰基氧 → 增強親電性 → H₂O 再進攻（機制 21.8，三步驟）

注意：催化劑只影響反應速率，不改變平衡。起始物若不穩定，催化劑也無法提高水合物的平衡比例。

11

醇的加成：縮醛形成

§21.14–21.15

縮醛（Acetal）的定義

同一個碳上連接兩個 OR 基（烷氧基），即 R₂C(OR')₂。

由醛或酮與2 當量醇在酸催化下反應，並脫去 1 當量水。

縮醛形成（酸催化）
R₂C=O + 2 R'OH ──TsOH（酸催化）──→ R₂C(OR')₂ + H₂O
環狀縮醛（用乙二醇）
酮 + HOCH₂CH₂OH ──TsOH──→ 環狀縮醛（五員環，含兩個 O）+ H₂O
縮醛 ≠ 醚：縮醛同一碳上有兩個 C–O 鍵，使其化學性質迥異於普通醚（只有一個 C–O–C）。

機制（兩段式）：半縮醛 → 縮醛

Mechanism 21.9 — 縮醛形成
1
半縮醛（hemiacetal）形成：酸先質子化羰基 → R'OH 進攻 → 去質子化 → 生成半縮醛（同一碳上有 OH 和 OR'）
R₂C=O + R'OH → R₂C(OH)(OR')（半縮醛）
2
半縮醛 → 縮醛：質子化 OH → 失去 H₂O（形成共振穩定碳陽離子）→ 第二個 R'OH 進攻 → 去質子化 → 生成縮醛
R₂C(OH)(OR') + R'OH → R₂C(OR')₂ + H₂O

縮醛形成是可逆反應，通常平衡偏向起始物（羰基+醇）。
製備縮醛時必須移除生成的水（Le Chatelier 原理），例如使用 Dean–Stark 脫水裝置，強迫平衡向產物方向移動。

Figure 21.11 說明
Dean–Stark 脫水裝置

原理：羰基化合物、醇和酸催化劑溶於苯中加熱。苯和水會共同揮發（共沸），冷凝液流入 Dean–Stark 管中。水密度較大沉在下層，可從旋塞放出。苯回流至反應瓶繼續循環。不斷移除水使反應右移，提高縮醛產率。

縮醛作為保護基（§21.15）

保護羰基的策略

合成含多個官能基的分子時，有時需要暫時保護羰基，使其不參與某步反應：

  1. 用乙二醇 + TsOH 將羰基轉換成環狀縮醛（保護）
  2. 進行其他化學反應（縮醛在中性或鹼性條件穩定！）
  3. 用稀酸水解縮醛 → 恢復羰基（去保護）

縮醛不與強還原劑（NaBH₄、LiAlH₄）、Grignard 試劑反應，因此可選擇性保護羰基。

12

環狀半縮醛

§21.16

當分子中同時含有 OH 和 C=O 基，且距離適合形成 5 或 6 員環時，可發生分子內環化，生成環狀半縮醛（lactol）。

環狀半縮醛的羰基碳上同時含有：
① 一個 –OH 基（來自環外）
② 一個 –OR 基（環內 O，即原來的醇 OH）

Figure — 環狀半縮醛數據
羥基醛的環化平衡

5-hydroxypentanal（6C 鏈）環化 → 六員環半縮醛：平衡有 94% 呈環狀
4-hydroxybutanal（5C 鏈）環化 → 五員環半縮醛：平衡有 89% 呈環狀
五員環和六員環的張力最小，因此這兩種環化反應特別有利。分子內反應（兩個官能基固定在同一分子）速率更快。

環化產生新的立體中心（原本的羰基碳），因此得到等量的兩種對映異構物（enantiomers）。

環狀半縮醛 → 環狀縮醛

環狀半縮醛加醇＋酸 → 把 OH 換成 OR → 環狀縮醛。
機制：OH 質子化 → 失去 H₂O → 共振穩定碳陽離子 → 醇進攻 → 去質子化。
重要：只有半縮醛的 OH 會反應，普通醇的 OH 不反應。

13

碳水化合物入門

§21.17

碳水化合物的定義

碳水化合物是含有多個 OH 基的醛或酮（polyhydroxy aldehydes and ketones），或是可水解成這類化合物的物質，俗稱糖類（sugars and starches）。

與蛋白質、脂肪酸、核苷酸並列為四大類生物大分子。

葡萄糖（glucose）與乳糖（lactose）中的半縮醛與縮醛
• 葡萄糖是血液中輸送到細胞的主要糖分。胰島素調節血糖濃度，糖尿病是胰島素分泌不足導致血糖異常升高的疾病。
• β-D-glucose 的環狀結構是一個六員環狀半縮醛，由開鏈的羥基醛自發環化形成。
• 乳糖（牛奶中的主要醣類）含有一個縮醛碳（醣苷鍵的碳）和一個半縮醛碳，體現了本章學習的縮醛化學。

糖類的半縮醛形成方式與實驗室中相同：羥基醛進行分子內環化。葡萄糖就是由含 6 個碳的羥基醛（開鏈式）環化而成六員環半縮醛。
