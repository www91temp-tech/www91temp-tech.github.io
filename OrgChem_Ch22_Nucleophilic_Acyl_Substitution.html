22.1

認識羧酸衍生物家族

本章主角是「醯基（acyl group）」——也就是 R–C(=O)– 接上不同電負性原子 Z 所形成的一系列化合物。

章節圖示說明：銀杏內酯 B（Ginkgolide B）是銀杏樹提取物中的主要成份，其緊密的 20 碳環系統在 1967 年被確定結構，1988 年由諾貝爾獎得主 Corey 完成全合成。該分子含有多個酯基——本章正是學習這類官能基反應的最佳舞台。

🔑 為什麼它們都能發生親核取代？

因為它們的羰基碳上都連接了一個電負性原子 Z，Z 可以充當離去基（leaving group）。
這和醛、酮不同——醛 / 酮沒有離去基，所以只能發生加成，不能取代。

每一個以羰基化合物為起始物的反應，都是親核醯基取代（Nucleophilic Acyl Substitution）。
22.2

結構與鍵結

🔺 羰基碳的兩大特性

  • sp² 混成、平面三角形：鍵角約 120°，空間位阻小 → 容易被親核試劑攻擊
  • C 帶 δ+ 正電：O 電負性強，使羰基碳成為親電中心（electrophilic）

📐 共振穩定性與鹼度的關係

RCOZ 可以畫出 3 個共振結構（比醛 / 酮多一個），Z 的鹼性越強，共振貢獻越大，化合物越穩定。

Cl⁻ < RCO₂⁻ < OH⁻ ≈ OR'⁻ < NR'₂⁻ ↑弱鹼 = 好離去基 強鹼 = 差離去基↑

命名規則

衍生物的命名都以母體羧酸為基礎，只需改變字尾。

酸氯化物 Acid Chloride（RCOCl）

將 -ic acid → -yl chloride
將 -carboxylic acid → -carbonyl chloride

acetic acid → acetyl chloride (CH₃COCl) cyclohexanecarboxylic acid → cyclohexanecarbonyl chloride

酸酐 Anhydride（(RCO)₂O）

將 acid → anhydride（"酐"字代表失水）

acetic acid + acetic acid → acetic anhydride acetic acid + benzoic acid → acetic benzoic anhydride（字母順序）

酯 Ester（RCOOR'）

先說 R' 烷基，再說醯基 -ic acid → -ate

CH₃COOC₂H₅ = ethyl acetate（烷基在前，醯基在後）

醯胺 Amide（RCONR'₂）

字尾（-ic acid、-oic acid、-ylic acid）改 -amide
N 上的烷基前加 N-

CH₃CONH₂ = acetamide (1°) CH₃CONHCH₂CH₃ = N-ethylacetamide (2°) CH₃CON(CH₃)₂ = N,N-dimethylacetamide (3°)

腈 Nitrile（RC≡N）

以碳原子鏈命名，CN 那個 C 算進去，字尾加 -nitrile

CH₃CH₂CN = propanenitrile（3個C，不是2個！） C₆H₅CN = benzonitrile

環狀特例：Lactone & Lactam

環狀酯 = lactone；環狀醯胺 = lactam

5員環酯 = γ-lactone 6員環酯 = δ-lactone 4員環醯胺 = β-lactam（青黴素的核心！） 5員環醯胺 = γ-lactam
命名速記：酸氯化物（-yl chloride）、酸酐（anhydride）、酯（alkyl + -ate）、醯胺（-amide），全都從羧酸名稱改字尾，認識羧酸命名就能搞定全部！

22.7 ★★★ 最核心

親核醯基取代——核心機制

整章所有反應都遵循同一個兩步驟機制：

⚙️ 機制 22.1：親核醯基取代通用機制
1

親核攻擊（Nucleophilic Attack）

親核試劑（:Nu⁻）攻擊親電性的羰基碳，π 鍵斷裂，電子移到氧上，形成 sp³ 混成的四面體中間體（tetrahedral intermediate），氧帶負電。

R–C(=O)–Z + :Nu⁻ → R–C(O⁻)(Nu)–Z
2

離去基離開（Elimination of Leaving Group）

氧上的電子對重新形成 C=O π 鍵，Z⁻ 帶著電子對離去，恢復 sp² 羰基，取代完成。

R–C(O⁻)(Nu)–Z → R–C(=O)–Nu + :Z⁻
⚠️ 若親核試劑是中性分子（H₂O、ROH、NH₃），還需要額外的質子轉移步驟，但核心兩步驟不變。

怎麼判斷反應會不會發生？

關鍵法則：離去基 Z⁻ 必須是比親核試劑 Nu⁻ 更弱的鹼（即更好的離去基），反應才能進行。

判斷步驟：
  1. 找出起始物的離去基 Z⁻ 和攻擊的親核試劑 Nu⁻
  2. 查鹼性順序：Cl⁻ < RCO₂⁻ < OH⁻ ≈ OR'⁻ < NR₂⁻
  3. 若 Z⁻ 鹼性 < Nu⁻ 鹼性 → ✔ 反應進行
  4. 若 Z⁻ 鹼性 > Nu⁻ 鹼性 → ✘ 反應不進行
口訣：「活潑的可以往穩定走，穩定的不能往活潑走。」
酸氯化物（最活潑）可以變成酯、醯胺……但醯胺（最穩定）不能直接變回酯或酸氯化物。
22.8

酸氯化物的反應

🔵 酸氯化物（Acid Chloride） RCOCl

活性最強 ⭐⭐⭐⭐⭐

Cl⁻ 是最弱的鹼、最好的離去基，所以酸氯化物與幾乎所有親核試劑都能快速反應，常需加吡啶（pyridine）中和副產物 HCl。

親核試劑產物反應條件
RCO₂⁻（羧酸根 carboxylate anion）酸酐 (RCO)₂O直接反應
H₂O羧酸 RCOOH加吡啶
R'OH（醇）酯 RCOOR'加吡啶
NH₃（2 equiv）1° 醯胺 RCONH₂過量氨作鹼
R'NH₂（2 equiv）2° 醯胺 RCONHR'過量胺作鹼
R'₂NH（2 equiv）3° 醯胺 RCONR'₂過量胺作鹼
生活例子：DEET（防蚊液活性成分 N,N-二乙基間甲苯醯胺）就是用間甲苯酸氯化物與二乙胺反應製成的 3° 醯胺。
它不殺蟲——而是混淆昆蟲的感知，讓牠們無法追蹤人體散發的溫濕氣息。
醯胺反應特別說明：需要用2 等量的胺——第 1 等量充當親核試劑進行取代，第 2 等量充當鹼來中和副產物 HCl（否則 HCl 會把胺質子化，使其失去親核性）。
22.9

酸酐的反應

🟠 酸酐（Anhydride） (RCO)₂O

活性次強 ⭐⭐⭐⭐

酸酐含兩個羰基，親核攻擊發生在其中一個羰基碳，另一個羰基成為離去基的一部分（RCO₂⁻ 離去）。

親核試劑產物
H₂O2 × 羧酸 RCOOH
R'OH（醇）酯 RCOOR' + RCOOH（副產物）
NH₃ / 1° / 2° 胺（2 equiv）醯胺 RCONH₂ + RCOO⁻NH₄⁺

⚠️ 注意：酸酐不能轉變為酸氯化物（因為 RCO₂⁻ 鹼性 > Cl⁻，Cl⁻ 不夠好的親核試劑）。

💊
重要例子：乙酸酐的乙醯化反應（ Acetylation reaction）
將乙醯基轉移到 –OH 或 –NH₂ 上
阿斯匹靈（Aspirin） 撲熱息痛（Acetaminophen / Tylenol） 海洛因（Heroin）

三者皆由乙酸酐（acetic anhydride）進行乙醯化反應製得。
阿斯匹靈是水楊酸的 –OH 被乙醯化為酯。
撲熱息痛是對胺基苯酚的 –NH₂ 被乙醯化為醯胺。
嗎啡（morphine）的兩個 –OH 同時被乙醯化，得到二酯——海洛因。

Mechanism 22.4 說明：演示酸酐與 NH₃ 反應製造醯胺的三步驟：（1）NH₃ 親核攻擊形成四面體中間體；（2）質子轉移；（3）RCO₂⁻ 作為離去基離去，得到 1° 醯胺。
22.10

羧酸的反應

🔴 羧酸（Carboxylic Acid） RCOOH

需特殊條件 ⭐⭐⭐

羧酸是有機強酸，鹼性親核試劑（如 NH₃、⁻OH）會先發生酸鹼反應，而非親核取代。所以需要特殊試劑來「活化」羧酸。

Figure 22.3 說明：羧酸可進行五類親核醯基取代反應：
（1）用 SOCl₂ 製備酸氯化物；（2）加熱脫水形成環狀酸酐；（3）酸催化與醇製備酯（Fischer esterification）；
（4）NH₃ 後加熱製備醯胺；（5）用 DCC 試劑與胺製備醯胺。

① 製備酸氯化物：用 SOCl₂（氯化亞碸，thionyl chloride）

RCOOH + SOCl₂ → RCOCl + SO₂ + HCl 機制重點：SOCl₂ 先把 –OH 轉為更好的離去基 OSOCl，再由 Cl⁻ 取代

這是將穩定的羧酸轉化成活潑的酸氯化物的唯一方法。

② 製備環狀酸酐：二元酸加熱脫水

丁二酸（succinic acid） + Δ → 丁二酸酐（succinic anhydride）+ H₂O 鄰苯二甲酸（phthalic acid） + Δ → 鄰苯二甲酸酐 + H₂O 只有相鄰兩個羧基（可形成5或6員環）才能內酯化

③ 製備酯：Fischer Esterification（費歇爾酯化）

RCOOH + R'OH  ⇌  RCOOR' + H₂O
催化劑：H₂SO₄（酸催化）。此反應為可逆平衡，需用過量醇或蒸除水來推動平衡向右。

機制核心（6 步驟）：

Part 1 — 親核試劑 R'OH 的加成：
① 酸催化：羰基氧質子化 → 羰基碳更親電
② R'OH 親核攻擊 → 四面體中間體
③ 去質子化
Part 2 — 離去基 H₂O 的離去：
④ OH 質子化 → 變成好離去基 H₂O
⑤ 脫去水 → 氧陽離子中間體
⑥ 去質子化 → 酯產物
鹼性條件下羧酸不能發生 Fischer 酯化！因為 ⁻OH 會把 RCOOH 去質子化成 RCOO⁻（羧酸根），帶負電的羧酸根不會被親核試劑攻擊。

④ 製備醯胺：與胺直接反應需要 DCC 試劑

直接混合羧酸與胺 → 先發生酸鹼反應，生成銨鹽；需加熱 >100°C 才能脫水成醯胺（產率低）。

RCOOH + R'NH₂ → RCOO⁻ R'NH₃⁺ →（加熱Δ）→ RCONHR' + H₂O 更好的方法：加入 DCC（二環己基碳二亞胺，dicyclohexylcarbodiimide）
DCC 的作用：DCC 是脫水劑，它把羧酸的 –OH 轉化為更好的離去基（活化中間體 O-acylisourea），使胺得以在溫和條件下直接親核取代，快速生成醯胺。副產物為二環己脲（dicyclohexylurea）。DCC 法廣泛應用於胜肽鍵（醯胺鍵）的合成。
22.11

酯的反應

🟢 酯（Ester） RCOOR'

中等活性 ⭐⭐

酯可水解為羧酸，也可與胺反應生成醯胺。水解有酸催化和鹼促進兩條路徑。

① 酸催化水解（可逆）

RCOOR' + H₂O ⇌ RCOOH + R'OH (H₂SO₄催化) 機制是 Fischer 酯化的逆過程，用大量水推動平衡向右

② 鹼性水解（皂化，Saponification）——不可逆！

RCOOR' + ⁻OH → RCOO⁻ + R'OH 生成的羧酸根 RCOO⁻ 是共振穩定的，驅動平衡，使反應不可逆
皂化（Saponification）：拉丁文"sapo"意為肥皂。脂肪（三醯甘油酯）用鹼水解，生成甘油和脂肪酸鹽——即肥皂！這也是鹼促進酯水解的典型應用。
⁻OH 是促進劑（promoter），不是催化劑（catalyst）！
因為 ⁻OH 是親核試劑，參與反應後被消耗，不會再生，所以稱為「鹼促進」而非「鹼催化」。

③ 與胺反應生成醯胺

RCOOR' + NH₃ → RCONH₂ + R'OH RCOOR' + R''NH₂ → RCONHR'' + R'OH 醇（R'OH）是離去基，pKa ~15–18，比 NH₂⁻ 的鹼性弱，反應可行
🫀
應用：脂質水解（22.12）
三醯甘油的代謝

動物以三醯甘油（triacylglycerol）形式儲存能量——本質上是三個長鏈脂肪酸以酯鍵連接在甘油骨架上。體內的脂酶（lipase）催化這些酯鍵的水解，釋放甘油和脂肪酸，供細胞燃燒產能。

Figure 22.6 說明：脂肪酸硫酯（fatty acid thioester）與膽固醇反應，在酶催化下形成膽固醇酯（cholesteryl ester）。膽固醇以酯的形式儲存在脂蛋白顆粒中，透過血液循環。LDL（低密度脂蛋白，俗稱「壞膽固醇」）把膽固醇送到組織；HDL（高密度脂蛋白，「好膽固醇」）把膽固醇送回肝臟。LDL 中的膽固醇酯若過多，會沉積在動脈壁形成粥狀硬化斑塊（plaque），是心血管疾病的主因。
Olestra（合成脂肪代替品）：Olestra 是蔗糖（8個 –OH）被脂肪酸酯化的產物。因為分子太大，消化酶無法水解它，所以不被吸收——這讓它成為零卡路里的脂肪代替品，但也會引起腸胃不適。
22.13

醯胺的反應

🟣 醯胺（Amide） RCONR'₂

最穩定，需強力條件 ⭐

NR'₂⁻ 是最強的鹼、最差的離去基，所以醯胺最穩定，只有在強酸或強鹼、高溫下才能水解。

條件反應產物
酸催化（H₃O⁺, 加熱）水解RCOOH + R'₂NH₂⁺
鹼（⁻OH, 加熱）水解RCOO⁻ + R'₂NH
醯胺鍵（peptide bond）是蛋白質的骨幹。正因為醯胺最穩定，蛋白質才能在生理條件下維持結構不被隨意水解。
🦠
應用：β-内醯胺抗生素（22.14）
青黴素 / 頭孢菌素的作用機制

青黴素（Penicillin）和頭孢菌素（Cephalosporin）的核心結構是β-內醯胺環（β-lactam）——一個四員環的醯胺。

作用機制：細菌合成細胞壁時，需要酶來連接糖肽聚合物。β-內醯胺環因為環張力大，C=O 吸收頻率偏高（更活潑），能與此酶的活性位絲胺酸（serine）–OH 發生不可逆的親核醯基取代（醯化反應）。酶被永久阻斷，細胞壁無法合成，細菌死亡。

Figure 22.14 說明：β-內醯胺環（四員環）因環張力而高度活潑。細菌酶的 Ser–OH 攻擊其羰基碳，開環後形成穩定的共價中間體（酯鍵），酶被永久失活。
抗藥性問題：部分細菌產生 β-內醯胺酶（β-lactamase），能水解 β-內醯胺環，使抗生素失效。這是抗藥性的重要機制之一。
22.16–22.17

應用：合成纖維與生物醯化

🧵 尼龍 6,6（Nylon 6,6）——聚醯胺

由兩個六碳單體縮合聚合：

己二醯氯（adipoyl chloride, Cl-CO(CH₂)₄CO-Cl） + 己二胺（hexamethylenediamine, H₂N(CH₂)₆NH₂） → 尼龍 6,6 + HCl（縮合聚合物）

1938 年 DuPont 建廠生產，最初用於製作降落傘，後取代蠶絲廣泛應用。

Figure 22.5 說明：
天然纖維的化學本質：羊毛和蠶絲是蛋白質（多個醯胺鍵連接的胺基酸鏈）；棉花和亞麻是纖維素（葡萄糖單元以醚鍵連接的多糖）。聚醯胺的尼龍模仿了蛋白質的醯胺鍵結構，而聚酯的滌綸（Dacron）則模仿了天然酯的連接方式。

🧶 聚酯 PET（Dacron / Terylene / Mylar）

對苯二甲酸（terephthalic acid） + 乙二醇（ethylene glycol） → PET + H₂O（Fischer 酯化縮合）

可回收再製成環保購物袋、飲料瓶（1號塑膠）、衣物纖維。

⚗️
生物醯化：乙醯輔酶 A（Acetyl CoA）（22.17）
生命體中最重要的醯基轉移試劑

細胞內的親核醯基取代叫做醯基轉移反應（acyl transfer）。主要試劑是硫酯（thioester），通式 RCOSR'，其中 ⁻SR' 是比 ⁻OR' 更好的離去基（S 鹼性比 O 弱）。

最重要的硫酯：乙醯輔酶 A（acetyl CoA），結構為 CH₃CO–SCoA。

例：合成乙醯膽鹼（acetylcholine）
乙醯輔酶 A（acetyl CoA） + 膽鹼（choline）→ 乙醯膽鹼 + CoA–SH
乙醯膽鹼是神經細胞間傳遞訊號的神經傳導物質（neurotransmitter）。
22.18

腈（Nitrile）RC≡N

基本性質

  • 含碳–氮三鍵（C≡N），C 為 sp 混成，線型結構（180°）
  • C 的氧化態與羧酸衍生物相同 → 化學行為有相似之處
  • 沒有離去基 → 不發生親核取代，而是發生親核加成（nucleophilic addition）
  • IR 特徵：C≡N 在 ~2250 cm⁻¹；¹³C NMR：115–120 ppm
結構圖說明：腈的 C≡N 類似炔烴的 C≡C，sp 混成，180° 線型。氮的電負性使 C 帶 δ+ 電，親核試劑從碳端攻擊。兩個抗癌藥物 Letrozole（商品名 Femara）和 Anastrozole（Arimidex）都含有腈基，作為芳香化酶抑制劑，阻斷雌激素合成，用於治療雌激素陽性乳癌。

腈的製備

R–X（伯鹵烷）+ ⁻CN →（SN2）→ R–C≡N + X⁻ SN2 取代，碳鏈增加 1 個碳，形成新的 C–C 鍵！（不能用仲、叔鹵烷）

腈的三大反應

① 水解 → 羧酸
RC≡N + H₂O → RCOOH（酸催化） → RCOO⁻（鹼促進） 機制：先加成形成亞胺酸， 互變異構為醯胺，再水解

三個 C–N 鍵被三個 C–O 鍵取代。

② 還原 → 胺 or 醛
RC≡N + [1] LiAlH₄ / [2] H₂O → RCH₂NH₂ （1° 胺） RC≡N + [1] DIBAL-H / [2] H₂O → RCHO （醛）

LiAlH₄ 加兩當量 H⁻；DIBAL-H 只加一當量，中間體為亞胺（imine），再水解得醛。

③ 與 RMgX / RLi 反應 → 酮
RC≡N + [1] R'MgX / [2] H₂O → R–CO–R' （酮）

形成新的 C–C 鍵，中間體為亞胺，水解後得酮。

腈反應記憶表：
腈 + H₂O → 羧酸（多加 OH，脫掉 N）
腈 + LiAlH₄ → 1° 胺（N 留下，加兩個 H）
腈 + DIBAL-H → 醛（N 離去，換成一個 H）
腈 + RMgX → 酮（N 離去，換成 R，生成新 C–C 鍵）
