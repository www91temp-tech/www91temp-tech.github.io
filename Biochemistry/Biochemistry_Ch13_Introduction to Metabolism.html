Ch.13 細胞如何做「功」？

代謝（metabolism）就是細胞把「吃進來的能量」轉換成各種生命活動的過程。本章先建立六個貫穿全書的核心原則，之後每一節都是把其中一、兩個原則講清楚。

細胞是開放系統（open system）：不斷與外界交換物質和能量，因此能一邊維持自身高度有序的結構，一邊仍然遵守「宇宙亂度必然增加」的熱力學第二定律——只是把亂度丟給了周遭環境。

P1生物的能量轉換遵循熱力學定律。
P2自由能變化可以加成；放能反應可「偶聯」帶動吸能反應。
P3看似成千上萬的代謝反應，其實只屬於少數幾種化學反應類型。
P4ATP 是通用的「能量貨幣」，靠轉移磷醯基來供能。
P5氧化還原反應把燃料的能量存進 NADH，再用來製造 ATP。
P6細胞靠調節酵素的「數量」與「活性」來應變環境變化。

13.1 生物能量學與熱力學

Bioenergetics and Thermodynamics — 用三個物理量（G、H、S）量化反應「能不能自發進行」。

熱力學兩大定律

定律內容對生物的意義
第一定律能量守恆，只能轉換形式，不能被創造或消滅細胞不能無中生有製造能量，只能轉換能量形式
第二定律宇宙的亂度（entropy）總是增加細胞維持自身秩序，代價是把更多亂度排放到環境中

三個關鍵物理量

符號中英名稱白話意思
G自由能（free energy）反應中「可以拿來做功」的能量
H焓（enthalpy）反應系統的「熱含量」，取決於化學鍵
S熵（entropy）系統的混亂 / 隨機程度
ΔG = ΔH − TΔS 自由能變化 = 焓變化 − (絕對溫度 × 熵變化)

✓放能 vs 吸能：一次分清楚

ΔG 符號反應性質類比
放能（exergonic）負值釋放自由能，可自發進行下坡滾球
吸能（endergonic）正值需要輸入自由能推球上坡
放熱（exothermic）ΔH 為負釋出熱到環境—
吸熱（endothermic）ΔH 為正從環境吸熱—

標準自由能變化 ΔG′° 與平衡常數 K′eq

ΔG′° 其實只是平衡常數 K′eq 的另一種數學表達方式：

ΔG′° = −RT ln K′eq
K′eqΔG′°標準條件下反應方向
大於 1負值正向進行
等於 1零已達平衡
小於 1正值逆向進行

!最容易混淆：ΔG 不等於 ΔG′°

ΔG′°（標準自由能變化）ΔG（實際自由能變化）
條件所有反應物/產物濃度皆為 1 M、pH 7、25℃ 的假設條件細胞內「實際」的濃度條件
是否會變動固定常數，對某反應永遠不變隨反應物、產物濃度即時變動
誰決定反應方向？不能單獨決定唯一能判斷反應是否自發的依據

關係式：ΔG = ΔG′° + RT ln Q（Q 為質量作用比，即目前 [產物]/[反應物] 的比值）。 即使 ΔG′° 是正值，只要細胞不斷把產物移走（Q 很小），ΔG 仍可能是負值而讓反應照樣往前跑。

自由能可以「加總」→ 偶聯反應

如果兩個反應共用一個中間產物，可以把它們相加成一個淨反應，ΔG′° 也直接相加。這正是細胞用 ATP 水解「贊助」原本不利反應的原理：

  1. 葡萄糖 + Pi → 葡萄糖6-磷酸 + H₂O　　ΔG′° = +13.8 kJ/mol（不利，不會自發）
  2. ATP + H₂O → ADP + Pi　　ΔG′° = −30.5 kJ/mol（非常有利）
  3. 加總：ATP + 葡萄糖 → ADP + 葡萄糖6-磷酸　　ΔG′° = −16.7 kJ/mol（整體變成放能反應！）

這就是「偶聯（coupling）」：用一個超放能反應，去帶動一個原本不利的反應，這是細胞中最基本的供能策略（P2）。

13.2 化學邏輯與常見生化反應

Chemical Logic and Common Biochemical Reactions — 細胞裡成千上萬個反應，其實只用了 5 種基本套路（P3）。

先懂兩個化學基本功

均裂（homolytic cleavage）異裂（heterolytic cleavage）
電子分配兩邊各拿走一顆電子一邊拿走整對電子（較常見）
產物自由基（radical）碳陽離子 / 碳陰離子 / 氫陰離子等

親核基（nucleophile，英）：帶有多餘電子、喜歡「送」電子的官能基（如帶負電的氧、硫）。
親電基（electrophile，英）：電子不足、喜歡「搶」電子的官能基（如羰基碳、磷酸基的磷）。反應的本質，就是親核基攻擊親電基。

代謝反應的五大類型

  1. 形成／打斷 C—C 鍵：如醛醇縮合、克來森酯縮合、脫羧反應——通常靠鄰近的羰基穩定不穩定的碳陰離子中間體。
  2. 分子內重排、異構化與脫去反應：原子不變，只是電子或雙鍵位置搬家，整體氧化態不變（如葡萄糖6-磷酸→果糖6-磷酸）。
  3. 自由基反應：涉及維生素 B12 或 SAM 等輔因子引發的均裂反應。
  4. 基團轉移反應：醯基、醣基、磷醯基從一個親核基轉移到另一個，磷醯基轉移在代謝中地位特別重要（見 13.3）。
  5. 氧化還原反應：電子從一個分子轉給另一個分子（見 13.4）。

這五類並非互斥——例如一個異構化反應內部也可能經過自由基中間體。

小提醒：生化方程式≠平衡的化學方程式

教科書常寫「ATP + H₂O → ADP + Pi」，這種生化方程式不會刻意平衡 H⁺ 或 Mg²⁺（因為它們的濃度視為不變）；若要嚴謹平衡電荷與原子，需要寫出完整的化學方程式（如 ATP⁴⁻ + H₂O → ADP³⁻ + HPO₄²⁻ + H⁺）。

13.3 磷醯基轉移與 ATP

Phosphoryl Group Transfers and ATP — ATP 是連結分解代謝與合成代謝的「共同貨幣」（P4）。

為什麼 ATP 水解會釋放大量自由能？

  1. Ⓟ電荷排斥被解除：ATP 上 4 個負電荷擠在一起，水解後分開，靜電排斥被釋放。
  2. Ⓟ產物共振穩定：釋出的 Pi 有多種共振結構可以分散電荷，比 ATP 本身更穩定。
  3. Ⓟ產物水合程度更高：ADP、Pi 比 ATP 更容易被水分子包圍穩定。
ATP⁴⁻ + H₂O → ADP³⁻ + HPO₄²⁻ + H⁺　　ΔG′° = −30.5 kJ/mol

高能 vs 低能磷酸化合物：一眼看懂排行

!最重要的觀念糾正：ATP 供能≠單純水解

課本常畫成一步「ATP→ADP+Pi」，但實際幾乎都是兩步驟的基團轉移：

常見誤解（一步）實際機制（兩步）
ATP 直接水解，釋放的熱去推動反應① 磷醯基先共價轉移到受質或酵素上（活化受質）
② 再由親核基取代，釋出 Pi

單純水解只會放熱，而熱在等溫系統中無法拿來做功——真正供能的是「磷醯基共價參與反應」。少數例外（如肌肉收縮、Na⁺/K⁺-ATPase）才是靠 ATP 直接水解驅動蛋白質構型改變。

ATP 的三大應用場景

應用機制重點
巨分子合成磷醯基／腺苷醯基轉移，活化前驅物，使其變成更好的親電基
主動運輸如 Na⁺K⁺-ATPase：磷醯化／去磷酸化循環，驅動蛋白質構型改變幫浦離子
肌肉收縮肌凝蛋白直接非共價結合並水解 ATP，驅動與肌動蛋白間的構型循環

維持 ATP 供應的三個酵素

酵素反應用途
核苷二磷酸激酶ATP + NDP ⇌ ADP + NTP把 ATP 的磷醯基分給 GTP、UTP、CTP 等
腺苷酸激酶2 ADP ⇌ ATP + AMP需求急升時緊急再生 ATP
肌酸激酶ADP + 磷酸肌酸 ⇌ ATP + 肌酸肌肉中快速補充 ATP 的緩衝庫

細胞內實際的 ATP 水解自由能（稱為磷酸化潛能 ΔGp）比標準值 −30.5 kJ/mol 更負（約 −50 kJ/mol），因為細胞會刻意把 [ATP] 維持在遠高於平衡濃度的地方。

13.4 生物氧化還原反應

Biological Oxidation-Reduction Reactions — 電子的流動，是所有生物做功的根本動力（P5）。

助記口訣

OIL RIG：Oxidation Is Losing（氧化＝失去電子）／Reduction Is Gaining（還原＝得到電子）

氧化還原反應可拆成兩個半反應（half-reaction）：一個失去電子（氧化），一個得到電子（還原）。生物體內常見的電子轉移方式包括：直接轉移電子、以氫原子形式轉移、以氫陰離子（:H⁻）形式轉移（NAD 家族）、或直接與氧結合。

還原電位：誰比較「想要」電子？

標準還原電位 E′°（單位：伏特）衡量一個氧化還原對搶電子的能力。E′° 越正，代表越容易被還原（越想搶電子）；電子會自發地從 E′° 較低流向 E′° 較高的一方。

ΔG′° = −n F ΔE′° n：轉移電子數；F：法拉第常數；ΔE′° = 電子接受者 E′° − 電子提供者 E′°

兩個半反應在還原電位表上距離越遠，電子流動釋放的自由能就越大——這正是粒線體電子傳遞鏈能一路釋能的原理。

兩大電子載體家族比較

主要角色分解代謝（catabolism）的氧化反應合成代謝（anabolism）的還原反應
細胞內常見狀態NAD⁺ 濃度高，利於接收氫陰離子NADPH 濃度高，利於供出氫陰離子
每次接受1 個氫陰離子（2 電子＋1 質子）同左
結合方式與酵素鬆散結合，可自由擴散同左

真實案例：菸鹼酸缺乏症——糙皮病

NAD⁺／NADP⁺ 皆由菸鹼酸（維生素 B3）合成。飲食中長期缺乏菸鹼酸會讓所有 NAD(P) 依賴的去氫酶失去功能，導致糙皮病（pellagra），典型症狀是「三D」：皮膚炎（Dermatitis）、腹瀉（Diarrhea）、失智（Dementia）。

13.5 代謝路徑的調節

Regulation of Metabolic Pathways — 沒有一個代謝反應能逃過調節（P6）。

動態穩態：濃度不變，流量卻可以很大

A →(v₁) S →(v₂) P 只要 v₁ = v₂，中間產物 [S] 的濃度就會維持恆定，即使反應「流量」很大也一樣

這就是恆定狀態（homeostasis）的數學本質——例如血糖濃度能穩定維持在約 5 mM，是因為「葡萄糖進入血液的速率」與「組織攝取葡萄糖的速率」互相配合。

調節酵素活性的兩個層次

層次方式時間尺度
調整「數量」轉錄因子調控基因表現，改變 mRNA / 蛋白質合成量數分鐘～數小時
調整蛋白質降解速率（如泛素化—蛋白酶體）數分鐘～數天
調整「活性」受質濃度變化（[S] 相對於 Km）即時
異位效應物結合（allosteric effector）毫秒級
共價修飾（如磷酸化／去磷酸化）數秒～數分鐘
與調節蛋白結合／解離依訊號而定

名詞辨析：代謝「調節」vs 代謝「控制」

代謝調節（regulation）代謝控制（control）
目的維持某個參數（如代謝物濃度）長期穩定因應外在訊號改變路徑的「輸出量」
例子血糖濃度維持在 5 mM 附近飢餓時啟動糖質新生、抑制糖解

平衡態 vs 遠離平衡態反應：調節的「熱點」在哪？

ATP／AMP：細胞的能量儀表板

[ATP] 下降一點點，經腺苷酸激酶反應（2ADP⇌ATP+AMP）會讓 [AMP] 大幅飆升（放大訊號）。AMP 活化蛋白激酶（AMPK） 偵測到 AMP 上升，就會促進葡萄糖攝取、糖解與脂肪酸氧化，同時抑制糖原、脂肪、膽固醇合成等耗能反應，目標是把 [ATP] 拉回來、[AMP] 壓下去。

🔑 重點複習 Key Takeaways

  • 01細胞是開放系統：靠不斷與環境交換物質與能量，在遵守熱力學第二定律的同時維持自身秩序。
  • 02判斷反應方向要看實際自由能變化 ΔG（= ΔG′° + RT ln Q），而不是固定不變的標準值 ΔG′°。
  • 03ΔG′° 具有加成性：放能反應（如 ATP 水解）可以「偶聯」帶動吸能反應，是細胞供能的核心策略。
  • 04成千上萬的代謝反應可歸納為 5 類：C—C 鍵形成/斷裂、分子內重排、自由基反應、基團轉移、氧化還原。
  • 05ATP 供能的本質是「磷醯基共價轉移」，幾乎不是單純水解；ATP 在能量位階上剛好介於高能與低能磷酸化合物之間，扮演能量轉運工。
  • 06氧化還原反應靠電子從低還原電位流向高還原電位，能量大小由 ΔG′° = −nFΔE′° 決定；NAD⁺ 管氧化分解代謝、NADPH 管還原合成代謝。
  • 07代謝路徑靠調節酵素的「數量」（合成/降解）與「活性」（受質濃度、異位效應、共價修飾、調節蛋白）維持動態穩態；遠離平衡的反應步驟才是主要調節點。
  • 08[ATP]/[AMP] 比值是細胞能量狀態最敏感的指標，AMPK 是回應這個訊號的關鍵調節器。
