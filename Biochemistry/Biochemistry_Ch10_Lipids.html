導論：什麼是脂質？

脂質 Lipids 是一群化學結構差異很大的分子，它們唯一的共同點就是不溶於水（疏水性）。 正因為「不溶於水」這個物理特性，脂質才能同時勝任脂肪儲存、細胞膜構造、訊號傳遞、色素、輔因子等看似毫不相關的工作。

本章不依化學結構分類，而是依照「脂質在細胞裡做什麼」來介紹，並歸納成下面四個核心原則（P1–P4）， 之後每一段落也會用相同顏色的徽章標示屬於哪一個原則，方便抓住脈絡。

P1 能量儲存

脂肪酸是高度還原的碳氫化合物，是細胞儲存化學能最有效率的形式；以三酸甘油酯儲存脂肪不需額外攜帶水分。

P2 膜的構造

膜脂質＝疏水尾巴＋親水頭部，這種兩性特質讓它們能自組裝成雙層膜，同時展示分子供辨識與訊號使用。

P3 訊號與其他功能

除了能量與膜構造，許多脂質是細胞內外的訊息分子：荷爾蒙、電子載體、色素等，量少但作用強大。

P4 結構決定性質

脂質的化學結構直接決定其物理性質；可用酵素分解、層析、質譜等方法鑑定脂質的原子結構。

10.1　儲存脂質 Storage Lipids

脂肪酸：碳氫化合物的衍生物 P1

脂肪酸 fatty acid 是帶有一個羧基（—COOH）的長鏈碳氫化合物，碳鏈長度從 4 到 36 個碳不等。可分為：

關鍵慣例 Key Convention｜脂肪酸的簡寫命名

「碳數：雙鍵數」，例如 16:0 表示 16 個碳、無雙鍵（棕櫚酸）；18:1 表示 18 個碳、1 個雙鍵（油酸）。

  • 羧基碳固定編號為 C-1，往後依序 C-2、C-3…
  • 雙鍵位置以 Δ（delta）加上較小的碳號表示，例如油酸雙鍵在 C-9–C-10 之間，記作 18:1(Δ⁹)。
關鍵慣例｜Omega（ω）命名法

從甲基端（離羧基最遠的那端）開始數，該碳定為 C-1（即 ω 碳）。

  • 雙鍵離甲基端第 3-4 個碳 → ω-3 脂肪酸（例如 EPA、DHA、α-次亞麻油酸 ALA）
  • 雙鍵離甲基端第 6-7 個碳 → ω-6 脂肪酸

人體無法自行合成 ALA，須從飲食攝取；地中海飲食富含 ω-3，被認為與較低心血管疾病風險有關。

三酸甘油酯：甘油的脂肪酸酯 P1

三酸甘油酯 triacylglycerol（俗稱中性脂肪、脂肪）由 一分子甘油（glycerol） 與 三分子脂肪酸 以酯鍵連接而成。 三個脂肪酸相同者稱「簡單三酸甘油酯」，不同者稱「混合三酸甘油酯」（自然界較常見）。

為什麼用脂肪而不是肝醣儲存能量？
  • 能量密度高：脂肪酸碳原子還原程度較高，氧化釋放的能量是等重醣類的兩倍以上。
  • 不需水合：脂肪疏水，不像肝醣每克需額外帶 2 克水，攜帶更「輕便」。
  • 代價：脂肪不易溶於水，動員速度較慢，因此醣類仍是快速能量來源。

氫化與反式脂肪：對健康的影響 醫學

植物油氧化後會變得油耗味（酸敗）。工業上以部分氫化將雙鍵轉為單鍵，可提升油品穩定性（如人造奶油）， 但過程中部分 cis 雙鍵會轉為 trans（反式）雙鍵。

醫學筆記｜反式脂肪與心血管疾病
  • 反式脂肪會提高血中三酸甘油酯與 LDL（壞膽固醇），降低 HDL（好膽固醇）
  • 可能增加身體發炎反應，是心臟病的額外風險因子
  • 目前各國法規已限制或禁止加工食品中使用反式脂肪

蠟：儲能兼防水 P1

蠟 wax 是長鏈脂肪酸與長鏈醇形成的酯，熔點比三酸甘油酯高（60–100 °C）。 浮游生物以蠟作為主要能量儲存形式；蜂蠟、羊毛脂、巴西棕櫚蠟等則廣泛用於化妝品與藥品產業，也是植物葉片防止水分過度蒸散的保護層。

10.1 小結
  • 脂肪酸多為偶數碳、不分支，雙鍵幾乎都是 cis 型
  • 三酸甘油酯＝甘油＋3 個脂肪酸酯鍵；主要用途是能量儲存與保溫
  • 脂解酶（lipase）水解三酸甘油酯釋出脂肪酸供燃燒使用
  • 反式脂肪來自部分氫化，與心血管疾病風險增加有關
  • 蠟＝長鏈脂肪酸＋長鏈醇，兼具儲能與防水功能

10.2　膜結構脂質 Structural Lipids in Membranes

P2膜脂質是兩性分子 amphipathic： 一端疏水、一端親水。這種特性讓它們在水中自然排列成雙層膜 bilayer， 疏水尾巴相對躲在內側，親水頭朝外接觸水。本節介紹四大類膜脂質： 磷脂質（phospholipid）、醣脂質（glycolipid）、 古菌四醚脂質（archaeal tetraether lipid）、固醇類（sterol）。

甘油磷脂質：磷脂酸的衍生物 P2

甘油磷脂質 glycerophospholipid（又稱磷酸甘油酯）： 甘油 C-1、C-2 接兩條脂肪酸（酯鍵），C-3 則透過磷酸雙酯鍵連接一個極性頭基，母體化合物為 磷脂酸 phosphatidic acid。頭基不同，就得到不同名稱的磷脂：

補充｜醚脂質 Ether Lipids

多數甘油磷脂以酯鍵連接脂肪酸，但少數（如縮醛磷脂 plasmalogen）以醚鍵連接， 對磷脂酶較有抵抗力。心臟組織中約一半的磷脂是縮醛磷脂；血小板活化因子則是另一種醚脂質，是重要的發炎訊號分子。

醣脂質與古菌脂質：環境的演化適應 P2

植物細胞的膜脂質以半乳糖脂 galactolipid為主（占維管束植物膜脂質 70–80%！）， 構造是二醯甘油接上 1–2 個半乳糖，不含磷酸——推測是植物為了節省土壤中珍貴的磷而演化出的替代方案。

極端環境古菌（如高溫、強酸、高鹽環境）的膜脂質則用醚鍵將長鏈（32 碳）分支烴類連接在甘油「兩端」， 分子長度是一般磷脂的兩倍，可直接橫跨整個細胞膜，耐熱耐酸性極佳。

鞘脂質：以鞘胺醇為骨架 P2

鞘脂質 sphingolipid不含甘油，而是以 鞘胺醇 sphingosine為骨架：C-2 的胺基以醯胺鍵接一條脂肪酸， C-1 則接不同的頭基。先接上脂肪酸、頭基還是氫原子的最簡單型態稱為神經醯胺 ceramide， 是所有鞘脂質的共同母體。

容易搞混：磷脂醯膽鹼 vs 神經鞘磷脂

兩者外觀、尺寸、電荷都非常相似，差別在骨架： 磷脂醯膽鹼骨架是甘油＋兩條酯鍵脂肪酸； 神經鞘磷脂骨架是鞘胺醇＋一條醯胺鍵脂肪酸。 這是判斷「甘油磷脂 vs 鞘磷脂」最快的方法。

人類血型（A、B、O）其實就是由紅血球細胞膜上醣鞘脂的寡醣頭基差異決定的——這些醣基同時也接在某些血液蛋白上， 所以「醣脂質決定血型」與「醣蛋白決定血型」兩種說法並不衝突。

醫學筆記｜鞘脂堆積症（溶小體儲積病）

鞘脂質在溶小體中被一系列專一酵素逐步分解。若某一酵素基因缺陷，中間產物會異常堆積：

神經鞘磷脂酶缺陷 → 神經鞘磷脂堆積 → Niemann-Pick 氏症：嬰兒智能障礙、早夭
己糖胺酶 A 缺陷 → 神經節苷脂 GM2 堆積 → Tay-Sachs 氏症：進行性發育遲緩、失明、多於 3–4 歲前死亡

可透過產前基因檢測（絨毛取樣、羊膜穿刺）提早發現風險。

磷脂酶：膜脂質的分解工具 P2

不同磷脂酶（phospholipase A、C、D）專一切割磷脂分子上不同的鍵結位置，是研究膜脂質結構的重要工具，也常被用於實驗室分析：

磷脂酶 A₁ / A₂：分別切除 C-1 / C-2 上的脂肪酸酯鍵，產生「溶血磷脂」
溶血磷脂酶：切除剩下的最後一條脂肪酸
磷脂酶 C：切割磷酸與甘油間的磷酸雙酯鍵 → 釋出水溶性頭基＋脂溶性二醯甘油
磷脂酶 D：切割磷酸與頭基之間的鍵

固醇類：四個相連的碳環 P2

固醇 sterol是幾乎所有真核細胞膜都有的結構脂質， 核心是類固醇骨架——三個六碳環＋一個五碳環，剛性且幾乎平面，環間 C—C 鍵無法自由旋轉。 膽固醇 cholesterol是動物細胞的代表性固醇：C-3 的羥基是極性頭， 其餘的環系統與側鏈是疏水部分，是典型的兩性分子。

固醇也是許多重要衍生物的前驅物：膽酸（bile acid）是膽固醇的極性衍生物，在腸道中作為清潔劑乳化脂肪，幫助消化酶作用； 類固醇荷爾蒙（見 10.3 節）也源自固醇骨架。植物有 stigmasterol、真菌有 ergosterol，但細菌無法自行合成固醇。

10.2 小結
  • 膜脂質皆為兩性分子：疏水尾＋親水頭，自組裝成雙層膜
  • 甘油磷脂＝甘油＋2 脂肪酸（酯鍵）＋磷酸雙酯頭基；心磷脂集中在粒線體內膜
  • 植物用半乳糖脂（無磷）、古菌用醚鍵長鏈脂質，都是環境/資源的演化適應
  • 鞘脂質以鞘胺醇為骨架，分神經鞘磷脂／中性醣脂／神經節苷脂三類；鞘脂分解異常會導致多種遺傳疾病
  • 固醇（如膽固醇）具四環剛性骨架，兼具膜結構與荷爾蒙前驅物角色

10.3　脂質作為訊號、輔因子與色素 Signals, Cofactors, Pigments

P3 這類脂質在細胞中含量很少，卻扮演「主動」角色：荷爾蒙、細胞內訊息分子、電子載體、光吸收色素等。

磷脂醯肌醇與鞘脂衍生物：細胞內訊號 P3

細胞膜內側的 PIP₂（磷脂醯肌醇 4,5-雙磷酸）是重要的「訊息儲存庫」：

細胞外荷爾蒙（如血管加壓素）與細胞表面受體結合
活化膜上的磷脂酶 C
磷脂酶 C 水解 PIP₂ → 產生 IP₃（水溶性）與二醯甘油（留在膜上）
IP₃ 觸發內質網釋放 Ca²⁺；二醯甘油＋Ca²⁺ 共同活化蛋白激酶 C
蛋白激酶 C 磷酸化下游蛋白質，產生細胞反應

此外，PIP₃可作為細胞膜上多蛋白複合體組裝的「停靠點」；鞘胺醇衍生物（神經醯胺、神經鞘磷脂）也能調控蛋白激酶，參與細胞分裂、分化與細胞凋亡。

類二十碳酸：旁分泌荷爾蒙 P3

類二十碳酸 eicosanoid 衍生自 20 碳的花生四烯酸（arachidonate）或 EPA，是旁分泌荷爾蒙——只作用於分泌來源附近的細胞。

醫學筆記｜NSAID 止痛藥的作用原理

阿斯匹靈、布洛芬等非類固醇消炎藥（NSAID）會抑制環氧化酶 COX， 阻斷花生四烯酸轉變為前列腺素與血栓素的路徑，因此能消炎、止痛、退燒。 每日低劑量阿斯匹靈則會促進脂氧素合成，用於預防心血管疾病。

類固醇荷爾蒙：長距離的基因調控訊號 P3

類固醇是固醇的氧化衍生物，失去 D 環上的烷基側鏈、極性較膽固醇高。它們透過血液（結合載體蛋白）運送到目標組織， 進入細胞後與核內受體蛋白結合，直接調控基因表現。代表性荷爾蒙：性荷爾蒙、腎上腺皮質分泌的皮質醇與醛固酮。 人工合成的普賴松（prednisone）則透過抑制磷脂酶 A₂（連帶抑制花生四烯酸釋放）達到強效抗發炎效果。

植物揮發性訊號 P3

植物會製造上千種脂溶性揮發物來吸引授粉者、驅趕草食動物或與其他植物「溝通」。例如由亞麻酸衍生的 茉莉酸（jasmonate）在昆蟲啃食時觸發防禦反應；牻牛兒醇（天竺葵香氣）、蒎烯（松樹）、檸檬烯（萊姆）皆屬此類， 許多也被用於香水工業。

脂溶性維生素：異戊二烯的家族 P3

維生素 KK₁（葉綠醌）、K₂（腸道菌合成）促凝血酶原活化，凝血必需；華法林可拮抗其作用（也是滅鼠藥／抗凝血藥）
醫學筆記｜黃金米與維生素A缺乏

全球每年約有 25 萬名兒童因維生素 A 缺乏而失明，且其中半數會在失明一年內死亡。 「黃金米計畫」透過基因工程讓稻米胚乳也能生成 β-胡蘿蔔素（維生素 A 前驅物），是重要的營養改善策略。

醌類與多元醇：電子載體與糖基載體 P3

輔酶 Q（泛醌 ubiquinone）在粒線體、質體醌（plastoquinone）在葉綠體中， 作為脂溶性電子載體參與 ATP 合成的氧化還原反應（詳見第19章）。 多元醇 dolichol則是異戊二烯醇，用來活化並「固定」醣基， 參與細菌細胞壁、醣蛋白與醣脂質的合成。

共軛雙烯色素與聚酮類天然物 P3

含多個交替單雙鍵（共軛雙烯）的脂質，因電子可離域，能吸收可見光而呈現顏色——如 β-胡蘿蔔素（橘黃）、 金絲雀羽毛的黃色色素等。聚酮類 polyketide則是次級代謝物， 合成路徑與脂肪酸類似，臨床上常作為抗生素（紅黴素）、抗真菌藥（兩性黴素 B）或降膽固醇藥物（洛伐他汀）。

10.3 小結
  • PIP₂ 水解產生 IP₃ ＋ 二醯甘油，是重要的細胞內雙訊息系統
  • 類二十碳酸（PG、TX、LT、LX）源自花生四烯酸，是強效旁分泌訊號，NSAID 可抑制其合成
  • 類固醇荷爾蒙經血液運輸、進核內調控基因表現
  • 維生素 A、D、E、K 皆為異戊二烯衍生物，各司視覺／鈣代謝／抗氧化／凝血功能
  • 輔酶Q、質體醌參與電子傳遞；多元醇負責糖基活化；聚酮類廣泛用於醫藥

10.4　脂質研究方法 Working with Lipids

P4因脂質不溶於水，其萃取與分離需要有機溶劑，方法也和蛋白質、醣類不同。

脂質分析標準流程

萃取：組織以氯仿／甲醇／水混合液勻漿，離心後分成兩相——氯仿相含脂質，甲醇/水相含蛋白質與醣類
分離：依極性差異，用吸附層析（矽膠管柱）或薄層層析 TLC分開脂質種類，極性愈高、洗脫愈晚
氣相層析 GC：先將脂肪酸轉為甲酯（衍生化）以提高揮發性，再依沸點/溶解度差異分離
專一水解：以特定磷脂酶（A、C、D）切割特定鍵結，產物再以 TLC/GC/HPLC 鑑定，推回原結構
質譜分析 MS：依碎片質荷比（m/z）確認碳鏈長度與雙鍵位置，或用「散彈法」直接分析未分離的粗萃取物
建構脂質體 Lipidome：整合上述數據，建立特定細胞/組織在特定條件下的完整脂質圖譜
薄層層析（TLC）vs 氣相層析（GC）vs 質譜（MS）
方法原理適合分析
TLC / 吸附層析依極性大小，於矽膠上以毛細作用分離快速分離脂質「類別」（如磷脂 vs 中性脂）
氣相層析 GC依揮發性／沸點差異，經惰性氣體帶動通過管柱分離不同鏈長、不同飽和度的脂肪酸衍生物
質譜 MS依碎片質荷比 m/z 鑑定分子結構精確確認碳鏈長度、雙鍵位置；可做「散彈法」全脂質體分析

LIPID MAPS 資料庫已建立一套系統性分類，將脂質分為八大化學類別（脂肪酸 FA、甘油酯 GL、甘油磷脂 GP、鞘脂 SP、 固醇脂 ST、異戊二烯醇脂 PR、糖脂 SL、聚酮類 PK），每種脂質都能以 12 碼「LM_ID」唯一識別—— 這與本章依「功能」分類的方式互補，各有優勢。

10.4 小結
  • 脂質萃取需用有機溶劑（如氯仿/甲醇/水），而非純水系統
  • TLC、吸附層析依極性分離；GC 依揮發性分離；兩者常搭配質譜做結構鑑定
  • 特定磷脂酶可切割特定鍵結，幫助推導未知脂質結構
  • 「散彈法」質譜可直接分析粗萃取物，加速「脂質體」全貌研究
