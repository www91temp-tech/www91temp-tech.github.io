什麼是醣類？

醣類（Carbohydrates）的分子式可以寫成 Cn(H₂O)n，是自然界中含量最豐富的有機化合物，在生物體中扮演多重角色：儲存能量、提供結構支撐、細胞辨識訊號等。

碳水化合物是多羥基的醛或酮（polyhydroxy aldehydes / ketones），或能水解得到這類化合物的物質。廣義包括：糖、澱粉、纖維素、甲殼素等。
📋 Figure 28.1 — 碳水化合物的多樣面貌
🍬
β-D-葡萄糖
最常見的單醣。血糖的主要來源，細胞最愛的即時能量。
🌲
纖維素 Cellulose
木材和植物莖的主要成分。葡萄糖以 β-糖苷鍵連接而成。
🧬

DNA 的四種核苷酸之一，醣的部份為去氧核糖。
💊
小紅莓黴素 Doxorubicin
抗癌藥物，結構中含有醣環。用於白血病、乳癌等治療。

光合作用 vs 代謝

碳水化合物是能量的「倉庫」，透過光合作用合成並儲存，再透過代謝釋放能量：

光合作用（儲能）：
6 CO₂ + 6 H₂O → hν/葉綠素 → C₆H₁₂O₆ (葡萄糖) + 6 O₂

代謝（釋能）：
C₆H₁₂O₆ + 6 O₂ → 6 CO₂ + 6 H₂O + 大量能量
雖然脂質每公克提供更多能量，但葡萄糖溶於水，可快速由血液輸送，在激烈運動需要爆發力時是首選。

單醣 Monosaccharides

單醣是最簡單的碳水化合物，含有 3～7 個碳，每個碳大多連接一個羥基。

兩大類單醣：
▸ 醛糖 Aldose：羰基（C=O）在 C1（醛基），如葡萄糖（D-glucose）
▸ 酮糖 Ketose：羰基（C=O）在 C2（酮基），如果糖（D-fructose）

立體化學：Fischer 投影式與 D / L 組態

Fischer 投影式規則

糖的立體中心常用「十字型」的 Fischer 投影式表示：

📋 Fischer 投影式的規則（圖示說明）
  • 1
    碳骨架垂直排列，醛基（CHO）或酮基置於頂端或靠近頂端。
  • 2
    水平鍵朝前方（楔形鍵，指向你），如同手臂往前伸。
  • 3
    垂直鍵朝後方（虛線楔形鍵，遠離你），如同腿往後退。
千萬不要在紙面上旋轉 Fischer 投影式！旋轉 90° 會把分子變成它的鏡像（對掌異構物）。

D / L 命名系統

比較分子中距離醛基最遠的立體中心上 OH 基的位置：
▸ OH 在右邊 → D-醣（自然界絕大多數醣為 D 型）
▸ OH 在左邊 → L-醣（較少見）

環狀結構：半縮醛的形成

單醣在水溶液中大部分以環狀半縮醛（hemiacetal）形式存在，而非開鏈形式。這是因為分子內的 OH 基攻擊羰基，形成穩定的環。

▸ 六員環醣（最常見）稱為 吡喃糖 Pyranose（C5 的 OH 環化）
▸ 五員環醣稱為 呋喃糖 Furanose（C4 的 OH 環化）
📋 Figure 28.6 — 葡萄糖的三種形式：α、β 及開鏈
互旋現象 Mutarotation：將任何一種純態的葡萄糖溶於水，最終達到平衡，混合物含 α(37%) + β(63%) + 開鏈(微量)。
原因：半縮醛可自由開環再環化，C1 每次環化都可能形成 α 或 β。

Haworth 投影式 — 立體畫法

📋 Figure — 如何從 Fischer 投影式轉換為 Haworth 投影式
  • 1
    畫六員環，氧原子置於右上角。
  • 2
    對 D-醣：CH₂OH 畫在環的上方。
  • 3
    決定異頭碳 anomeric carbon（C1）的構型：
    ▸ α 異頭物：OH 在下方（軸向 axial）（D-醣中與 CH₂OH 反式）
    ▸ β 異頭物：OH 在上方（赤道向 equatorial）（D-醣中與 CH₂OH 順式）
  • 4
    其餘立體中心：Fischer 投影式右側 → Haworth 下方；Fischer 左側 → Haworth 上方。
口訣：「右下左上，CH₂OH 朝上，氧在右上」——D-醣 Haworth 投影式的黃金口訣

椅形構型 — 最真實的 3D 結構

📋 Figure 28.7 — β-D-葡萄糖的椅形構型

β-D-葡萄糖是葡萄糖中最穩定的形式，因為所有大的取代基（OH、CH₂OH）都位於赤道向鍵（equatorial）位置，空間障礙最小。

葡萄糖是最普遍的單醣，因為它在椅形構型中所有取代基都偏好赤道向鍵，能量最低、最穩定！

糖苷 Glycosides

單醣的半縮醛 OH 基（異頭碳上的 OH）與醇反應（經酸催化），形成縮醛，稱為糖苷（glycoside）。

α-D-葡萄糖 + CH₃OH + HCl → α-甲基葡萄糖苷 + β-甲基葡萄糖苷
（半縮醛 → 縮醛，只有異頭 OH 會反應）
糖苷 vs 半縮醛的關鍵差異：
▸ 半縮醛（-OH）：在水中開環 ⇌ 可互旋 ⇌ 是還原糖（reducing sugars）
▸ 縮醛（-OR）= 糖苷：不開環，不互旋，是非還原糖

糖苷形成機構（Mechanism 28.1）

📋 Mechanism 28.1 — 糖苷如何形成
  • 1
    HCl 質子化半縮醛的 OH 基
  • 2
    脫去 H₂O，形成共振穩定的碳陽離子 carbocation（↔ 氧碳陽離子 oxocarbenium ion）
    ——環狀平面結構
  • 3
    CH₃OH 從平面上方或下方進行親核攻擊
  • 4
    去質子化，分別得到 β 和 α 兩種糖苷
只有異頭碳的 OH 才有此反應性，因為失去 OH 後能形成共振穩定的碳陽離子；其他 OH 失去後無法共振穩定，故不反應。

糖苷水解

糖苷（縮醛）+ H₃O⁺ + H₂O → 半縮醛（α + β 混合）+ 醇
機構與形成反應的逆過程完全相同。

8 OH 基的反應

醚化（甲基化）— 保護 OH 基

用 Ag₂O + CH₃I 或 NaH + CH₃I 將所有 OH 基甲基化。
普通的醚（非糖苷位）對酸穩定；但糖苷鍵（縮醛醚）可被酸水解。這個差異在結構鑑定上很有用。

酯化（乙醯化）

用 乙酸酐（Ac₂O）+ 吡啶將所有 OH 基轉為乙酸酯。
目的：單醣極性大，難溶於有機溶劑；乙醯化後可溶於有機溶劑，方便操作。

雙醣 Disaccharides

雙醣由兩個單醣環以糖苷鍵（glycosidic linkage）連接。糖苷鍵由一個環的異頭碳（C1）連到另一個環上的任意 OH。

三大重要雙醣對比

📋 Figure 28.9 相關 — 麥芽糖、乳糖、蔗糖比較
雙醣 組成單醣 糖苷鍵 還原性 互旋 來源/備註
麥芽糖 Maltose 葡萄糖 + 葡萄糖 1→4-α ✅ 還原糖 ✅ 有 澱粉水解產物；麥芽汁
乳糖 Lactose 半乳糖 + 葡萄糖 1→4-β ✅ 還原糖 ✅ 有 牛奶；乳糖不耐症與 β-半乳糖苷酶相關
蔗糖 Sucrose 葡萄糖 + 果糖 α,β-1,2（兩個異頭碳都用） ❌ 非還原糖 ❌ 無 蔗糖、甜菜根；最常見雙醣
蔗糖為什麼是非還原糖？
因為葡萄糖和果糖的兩個異頭碳都參與形成糖苷鍵，沒有任何半縮醛 OH 殘餘，所以完全無法開環形成醛基，也就無法還原 Tollens/Benedict 試劑。
人工甜味劑（Figure 28.10）：
蔗糖的甜度讓人愉悅，但熱量高。三大人工甜味劑發現趣聞：
• 蔗糖素（Splenda）：1976 年化學家誤嘗化合物而發現
• 阿斯巴甜（Equal）：1965 年研究員舔了髒手指而發現
• 糖精（Sweet'n Low）：1879 年化學家忘記洗手而發現
——三種都是「意外的甜蜜」😄

13B N-糖苷 → 核苷 → 核苷酸 → DNA/RNA

📋 Figure 28.11 — RNA 與 DNA 的醣骨架（核心概念！）

糖 + 鹼（胺類）→ N-糖苷（核苷 Nucleoside）
異頭碳（C1）與含氮鹼基（N 原子）連接，形成 β-N-糖苷鍵。

核苷 + 磷酸 → 核苷酸 Nucleotide
醣上的 OH 與磷酸酯化，得到核苷酸（RNA/DNA 的建構單元）。

核苷酸聚合 → RNA 或 DNA
各核苷酸以磷酸二酯鍵（3'→5'）串接，形成核酸主鏈。

核酸 醣基 鹼基 功能
RNA D-核糖（C2 有 OH） A, G, C, U（尿嘧啶） 基因訊息傳遞 → 蛋白質合成
DNA 2-去氧-D-核糖（C2 無 OH） A, G, C, T（胸腺嘧啶） 遺傳訊息的永久儲存
RNA vs DNA 的醣差異記憶法：
DNA 的「去氧」= C2位置「去掉」一個氧（OH → H）
RNA 保有 C2-OH → 更容易水解 → 適合短期傳訊

重要反應總整理（點擊展開）

氧化反應 — 如何判斷還原糖？

用 Tollens 試劑（Ag₂O/NH₄OH）或 Benedict/Fehling 試劑（Cu²⁺）測試：

  • 出現銀鏡或磚紅色 Cu₂O 沉澱 → 含自由醛基 → 還原糖
  • 無反應 → 非還原糖（純縮醛，如糖苷或蔗糖）
環化規則 — 哪個 OH 環化？
  • 醛己糖（如葡萄糖）：C5 的 OH 環化 → 六員吡喃環
  • 醛戊糖（如核糖）：C4 的 OH 環化 → 五員呋喃環
  • 酮己糖（如果糖）：C5 的 OH 環化 → 五員呋喃環（因為 C2 是羰基）
糖苷鍵命名 — 如何讀？

以「1→4-α-糖苷鍵」為例：

  • 1 = 一個單醣的異頭碳（C1）
  • 4 = 連到另一個單醣的 C4
  • α = 異頭碳的構型（α=下，β=上，D-醣）

例：麥芽糖 = 1→4-α；乳糖 = 1→4-β；蔗糖 = 1,2-α,β

立體異構物數目計算

立體中心數 n → 最多 2ⁿ 種立體異構物：

  • 醛三糖（甘油醛）：1 個立體中心 → 2¹ = 2 種（1 D + 1 L）
  • 醛四糖：2 個立體中心 → 2² = 4 種
  • 醛戊糖：3 個立體中心 → 2³ = 8 種
  • 醛己糖：4 個立體中心 → 2⁴ = 16 種（8 D + 8 L）
  • 環化後葡萄糖：5 個立體中心（+異頭碳）→ 32 種可能
Fischer 投影式 → Haworth 投影式 → 椅形構型 轉換步驟

Fischer → Haworth：

  • ①畫六員環，O在右上角
  • ② D-醣：CH₂OH 畫上方
  • ③ α異頭物：C1-OH 向下；β異頭物：C1-OH 向上
  • ④ Fischer 右側 → Haworth 下方；Fischer 左側 → Haworth 上方

Haworth → 椅形：

  • O 原子作為「上仰」位置
  • Haworth 上方取代基 → 椅形上鍵；Haworth 下方 → 椅形下鍵
  • 判斷是軸向鍵還是赤道向鍵取決於碳的位置
